# root cause analysis

RCA can be decomposed into {@{4 steps}@}: {@{problem, trace, label, and graph}@}. <!--SR:!2028-09-07,1206,350!2027-11-30,978,341-->

1. problem ::@:: Identify and describe the problem clearly. <!--SR:!fsrs,2032-01-27T04:16:35.054Z,2054,2053.88360027,1,2,10,0,0,2026-06-13T04:16:35.054Z!2028-07-15,1165,350-->
2. trace ::@:: Establish a timeline from a normal situation to the problem. <!--SR:!2028-12-26,1292,350!2027-03-16,759,330-->
3. label ::@:: Distinguish between the root causes and other casual factors. <!--SR:!2026-07-16,588,330!fsrs,2031-11-02T00:00:00.000Z,1950,1949.97677109,1,2,10,0,0,2026-07-01T00:00:00.000Z-->
4. graph ::@:: Establish a [casual graph](casual%20graph.md) from the root causes to the problem. <!--SR:!2028-03-08,999,310!2027-12-04,991,350-->
