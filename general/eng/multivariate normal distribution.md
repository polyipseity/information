In {@{[probability theory](probability%20theory.md) and [statistics](statistics.md)}@}, {@{the __multivariate normal distribution__, __multivariate Gaussian distribution__, or __joint normal distribution__}@} is {@{a generalization of the one-dimensional \([univariate](univariate.md)\) [normal distribution](normal%20distribution.md) to higher [dimensions](dimension.md)}@}. {@{One definition}@} is that {@{a [random vector](random%20vector.md) is said to be _k_-variate normally distributed}@} if {@{every [linear combination](linear%20combination.md) of its _k_ components has a univariate normal distribution}@}. Its importance derives mainly from {@{the [multivariate central limit theorem](central%20limit%20theorem.md#multidimensional%20CLT)}@}. {@{The multivariate normal distribution}@} is often used to describe, {@{at least approximately, any set of \(possibly\) [correlated](correlation%20(statistics).md) real-valued [random variables](random%20variable.md)}@}, each of which {@{clusters around a mean value}@}. <!--SR:!2026-04-14,67,337!2026-05-09,88,359!2026-04-22,71,322!2026-05-13,90,362!2026-05-10,89,362!2026-04-27,76,337!2026-05-05,83,348!2026-04-27,76,337!2026-05-09,86,362!2026-05-06,84,348-->

## definitions

### notation and parametrization

{@{The multivariate normal distribution of a _k_-dimensional random vector $\mathbf {X} =(X_{1},\ldots ,X_{k})^{\mathrm {T} }$}@} can be written in the following notation: {@{$$\mathbf {X} \ \sim \ {\mathcal {N} }({\boldsymbol {\mu } },\,{\boldsymbol {\Sigma } }),$$}@} or to make it {@{explicitly known that $\mathbf {X}$ is _k_-dimensional}@}, {@{$$\mathbf {X} \ \sim \ {\mathcal {N} }_{k}({\boldsymbol {\mu } },\,{\boldsymbol {\Sigma } }),$$}@} with {@{_k_-dimensional [mean vector](mean%20vector.md)}@} {@{$${\boldsymbol {\mu } }=\operatorname {E} [\mathbf {X} ]=(\operatorname {E} [X_{1}],\operatorname {E} [X_{2}],\ldots ,\operatorname {E} [X_{k}])^{\mathrm {T} },$$}@} and {@{$k\times k$ [covariance matrix](covariance%20matrix.md)}@} {@{$$\Sigma _{i,j}=\operatorname {E} [(X_{i}-\mu _{i})(X_{j}-\mu _{j})]=\operatorname {Cov} [X_{i},X_{j}]$$}@} such that {@{$1\leq i\leq k$ and $1\leq j\leq k$}@}. {@{The [inverse](matrix%20inverse.md) of the covariance matrix}@} is called {@{the [precision matrix](precision%20(statistics).md)}@}, denoted by {@{${\boldsymbol {Q} }={\boldsymbol {\Sigma } }^{-1}$}@}. <!--SR:!2026-04-19,71,337!2026-05-11,88,362!2026-05-14,91,362!2026-05-06,84,348!2026-05-13,92,362!2026-03-21,45,328!2026-04-28,77,337!2026-05-13,92,362!2026-05-09,88,362!2026-04-30,78,348!2026-04-28,77,337!2026-05-09,88,362-->

### standard normal random vector

{@{A real [random vector](random%20vector.md) $\mathbf {X} =(X_{1},\ldots ,X_{k})^{\mathrm {T} }$}@} is called {@{a __standard normal random vector__}@} if {@{all of its components $X_{i}$ are independent}@} and each is {@{a zero-mean unit-variance normally distributed random variable}@}, i.e. if {@{$X_{i}\sim \ {\mathcal {N} }(0,1)$ for all $i=1\ldots k$}@}.<sup>[\[1\]](#^ref-1)</sup><sup>:&hairsp;p. 454&hairsp;</sup> <!--SR:!2026-04-26,77,337!2026-04-22,71,322!2026-05-13,90,362!2026-05-15,92,359!2026-04-20,72,337-->

### centered normal random vector

{@{A real random vector $\mathbf {X} =(X_{1},\ldots ,X_{k})^{\mathrm {T} }$}@} is called {@{a __centered normal random vector__}@} if there exists {@{a $k\times \ell$ matrix ${\boldsymbol {A} }$}@} such that {@{${\boldsymbol {A} }\mathbf {Z}$ has the same distribution as $\mathbf {X}$}@} where $\mathbf {Z}$ is {@{a standard normal random vector with $\ell$ components}@}.<sup>[\[1\]](#^ref-1)</sup><sup>:&hairsp;p. 454&hairsp;</sup> <!--SR:!2026-04-28,77,337!2026-05-06,84,348!2026-05-08,86,362!2026-04-28,79,348!2026-05-06,84,348-->

### normal random vector

{@{A real random vector $\mathbf {X} =(X_{1},\ldots ,X_{k})^{\mathrm {T} }$}@} is called {@{a __normal random vector__}@} if there exists {@{a random $\ell$-vector $\mathbf {Z}$, which is a standard normal random vector}@}, {@{a $k$-vector ${\boldsymbol {\mu } }$, and a $k\times \ell$ matrix ${\boldsymbol {A} }$}@}, such that {@{$\mathbf {X} ={\boldsymbol {A} }\mathbf {Z} +{\boldsymbol {\mu } }$}@}.<sup>[\[2\]](#^ref-2)</sup><sup>:&hairsp;p. 454&hairsp;</sup><sup>[\[1\]](#^ref-1)</sup><sup>:&hairsp;p. 455&hairsp;</sup> <!--SR:!2026-05-14,91,362!2026-05-02,80,348!2026-04-15,68,335!2026-05-14,91,362!2026-05-04,84,348-->

Formally:

> (annotation: definition of normal random vector) ::@:: $$\mathbf {X} \ \sim \ {\mathcal {N} }_{k}({\boldsymbol {\mu } },{\boldsymbol {\Sigma } })\iff {\text{there exist } }{\boldsymbol {\mu } }\in \mathbb {R} ^{k},{\boldsymbol {A} }\in \mathbb {R} ^{k\times \ell }{\text{ such that } }\mathbf {X} ={\boldsymbol {A} }\mathbf {Z} +{\boldsymbol {\mu } }{\text{ and } }\forall n=1,\ldots ,\ell :Z_{n}\sim \ {\mathcal {N} }(0,1),{\text{i.i.d.} }$$ <!--SR:!2026-05-01,81,348!2026-05-15,92,362-->

Here {@{the [covariance matrix](covariance%20matrix.md)}@} is {@{${\boldsymbol {\Sigma } }={\boldsymbol {A} }{\boldsymbol {A} }^{\mathrm {T} }$}@}. <!--SR:!2026-04-26,75,337!2026-05-04,82,348-->

In {@{the [degenerate](degeneracy%20(mathematics).md) case where the covariance matrix is [singular](singular%20matrix.md)}@}, the corresponding distribution has {@{no density}@}; see the [section below](#degenerate%20case) for details. This case arises {@{frequently in [statistics](statistics.md)}@}; for example, in the distribution of {@{the vector of [residuals](errors%20and%20residuals%20in%20statistics.md)}@} in {@{the [ordinary least squares](ordinary%20least%20squares.md) regression}@}. {@{The $X_{i}$}@} are {@{in general _not_ independent}@}; they can be seen as the result of applying {@{the matrix ${\boldsymbol {A} }$ to a collection of independent Gaussian variables $\mathbf {Z}$}@}. <!--SR:!2026-04-24,73,322!2026-04-26,75,337!2026-04-28,77,337!2026-04-12,65,322!2026-05-10,87,359!2026-05-12,91,359!2026-04-26,77,337!2026-05-13,90,362-->

### equivalent definitions

The following definitions are {@{equivalent to the definition given above}@}. {@{A random vector $\mathbf {X} =(X_{1},\ldots ,X_{k})^{\mathrm {T} }$}@} has {@{a multivariate normal distribution}@} if it {@{satisfies one of the following equivalent conditions}@}. \(annotation: 2 items: linear combinations, characteristic function\) <!--SR:!2026-04-17,67,310!2026-04-17,69,322!2026-05-15,92,362!2026-04-30,78,348-->

- {@{Every linear combination $Y=a_{1}X_{1}+\cdots +a_{k}X_{k}$ of its components}@} is {@{[normally distributed](normal%20distribution.md)}@}. That is, for {@{any constant vector $\mathbf {a} \in \mathbb {R} ^{k}$}@}, {@{the random variable $Y=\mathbf {a} ^{\mathrm {T} }\mathbf {X}$}@} has {@{a univariate normal distribution}@}, where {@{a univariate normal distribution with zero variance}@} is {@{a point mass on its mean}@}.
- There is {@{a _k_-vector $\mathbf {\mu }$ and a symmetric, [positive semidefinite](positive-definite%20matrix.md#Negative-definite.2C%20semidefinite%20and%20indefinite%20matrices) $k\times k$ matrix ${\boldsymbol {\Sigma } }$}@}, such that {@{the [characteristic function](characteristic%20function%20(probability%20theory).md) of $\mathbf {X}$}@} is {@{$$\varphi _{\mathbf {X} }(\mathbf {u} )=\exp {\Big (}i\mathbf {u} ^{\mathrm {T} }{\boldsymbol {\mu } }-{\tfrac {1}{2} }\mathbf {u} ^{\mathrm {T} }{\boldsymbol {\Sigma } }\mathbf {u} {\Big )}.$$}@} {@{The spherical normal distribution}@} can be characterised as {@{the unique distribution}@} where components are {@{independent in any orthogonal coordinate system}@}.<sup>[\[3\]](#^ref-3)</sup><sup>[\[4\]](#^ref-4)</sup> <!--SR:!2026-04-23,75,348!2026-05-15,92,362!2026-04-27,76,337!2026-05-12,89,359!2026-05-12,91,362!2026-05-07,86,362!2026-05-02,80,348!2026-05-14,91,359!2026-05-15,92,362!2026-03-10,34,297!2026-05-12,91,362!2026-05-15,92,359!2026-04-19,71,337-->

#### non-degenerate case

{@{The multivariate normal distribution}@} is said to be {@{"non-degenerate"}@} when {@{the symmetric [covariance matrix](covariance%20matrix.md) ${\boldsymbol {\Sigma } }$ is [positive definite](positive-definite%20matrix.md)}@}. In this case the distribution has {@{[density](probability%20density%20function.md)}@}<sup>[\[5\]](#^ref-5)</sup> {@{$$f_{\mathbf {X} }(x_{1},\ldots ,x_{k})={\frac {\exp \left(-{\frac {1}{2} }\left({\mathbf {x} }-{\boldsymbol {\mu } }\right)^{\mathrm {T} }{\boldsymbol {\Sigma } }^{-1}\left({\mathbf {x} }-{\boldsymbol {\mu } }\right)\right)}{\sqrt {(2\pi )^{k}|{\boldsymbol {\Sigma } }|} } }$$}@} where ${\mathbf {x} }$ is {@{a real _k_-dimensional column vector}@} and $|{\boldsymbol {\Sigma } }|\equiv \det {\boldsymbol {\Sigma } }$ is {@{the [determinant](determinant.md) of ${\boldsymbol {\Sigma } }$, also known as the [generalized variance](generalized%20variance.md)}@}. {@{The equation above}@} reduces to that of {@{the univariate normal distribution}@} if {@{${\boldsymbol {\Sigma } }$ is a $1\times 1$ matrix \(i.e., a single real number\)}@}. <!--SR:!2026-05-13,90,362!2026-04-21,73,337!2026-05-05,83,348!2026-05-11,88,362!2026-03-20,42,322!2026-05-03,81,348!2026-05-14,91,362!2026-05-08,87,359!2026-04-26,75,337!2026-05-08,87,362-->

{@{The circularly symmetric version of the [complex normal distribution](complex%20normal%20distribution.md)}@} has {@{a slightly different form}@}. <!--SR:!2026-05-09,86,362!2026-04-23,75,348-->

Each {@{iso-density [locus](locus%20(mathematics).md)}@} — the locus of {@{points in _k_-dimensional space each of which gives the same particular value of the density}@} — is {@{an [ellipse](ellipse.md) or its higher-dimensional generalization}@}; hence {@{the multivariate normal}@} is a special case of {@{the [elliptical distributions](elliptical%20distribution.md)}@}. <!--SR:!2026-05-13,90,362!2026-04-16,69,337!2026-05-14,91,359!2026-05-14,91,362!2026-05-13,90,359-->

{@{The quantity ${\sqrt {({\mathbf {x} }-{\boldsymbol {\mu } })^{\mathrm {T} }{\boldsymbol {\Sigma } }^{-1}({\mathbf {x} }-{\boldsymbol {\mu } })} }$}@} is known as {@{the [Mahalanobis distance](Mahalanobis%20distance.md)}@}, which represents {@{the distance of the test point ${\mathbf {x} }$ from the mean ${\boldsymbol {\mu } }$}@}. {@{The squared Mahalanobis distance $({\mathbf {x} }-{\boldsymbol {\mu } })^{\mathrm {T} }{\boldsymbol {\Sigma } }^{-1}({\mathbf {x} }-{\boldsymbol {\mu } })$}@} is decomposed into {@{a sum of _k_ terms}@}, (annotation: The decomposition is: {@{$$\begin{aligned} & \phantom = (x - \mu)^T \Sigma^{-1} (x - \mu) \\ & = (x - \mu)^T P \Lambda^{-1} P^T (x - \mu) \\ & = (P^T (x - \mu))^T \Lambda^{-1} (P^T (x - \mu)) \\ & = \sum_{i=1}^k \frac{z_i^2}{\lambda_i} \,, \end{aligned}$$}@} where {@{$P$}@} is {@{the matrix of eigenvectors of $\Sigma$, which defines the principal directions}@}. {@{$\Lambda = \text{diag}(\lambda_1, \dots, \lambda_k)$}@} is {@{the diagonal matrix of eigenvalues, representing the variances along those directions}@}. {@{The term $z = P^T(x - \mu)$}@} gives {@{the coordinates of $(x - \mu)$ in the principal component space}@}.) each term being {@{a product of three meaningful components}@}.<sup>[\[6\]](#^ref-6)</sup> (annotation: It represents {@{the sum of squared deviations along each principal axis}@}, scaled by {@{the inverse of its variance}@}. It measures {@{how far the point is}@} from {@{the mean in standardized units across all directions}@}.) Note that in the case when {@{$k=1$}@}, {@{the distribution}@} reduces to {@{a univariate normal distribution}@} and {@{the Mahalanobis distance}@} reduces to {@{the absolute value of the [standard score](standard%20score.md)}@}. See also [Interval](#interval) below. <!--SR:!2026-04-03,55,310!2026-04-25,76,335!2026-05-14,91,362!2026-04-26,75,337!2026-04-09,57,328!2026-05-15,92,362!2026-04-13,66,322!2026-05-13,90,359!2026-05-06,84,348!2026-05-14,91,359!2026-04-10,58,328!2026-04-28,77,337!2026-04-23,75,348!2026-05-14,91,359!2026-04-22,73,322!2026-04-28,77,337!2026-05-09,86,362!2026-05-15,92,362!2026-05-15,92,362!2026-05-12,91,362!2026-04-25,74,322!2026-05-04,84,348-->

#### bivariate case

In {@{the 2-dimensional nonsingular case \($k=\operatorname {rank} \left(\Sigma \right)=2$\)}@}, {@{the [probability density function](probability%20density%20function.md) of a vector ${\text{[XY]} }\prime$}@} is: {@{$$f(x,y)={\frac {1}{2\pi \sigma _{X}\sigma _{Y}{\sqrt {1-\rho ^{2} } } } }\exp \left(-{\frac {1}{2\left[1-\rho ^{2}\right]} }\left[\left({\frac {x-\mu _{X} }{\sigma _{X} } }\right)^{2}-2\rho \left({\frac {x-\mu _{X} }{\sigma _{X} } }\right)\left({\frac {y-\mu _{Y} }{\sigma _{Y} } }\right)+\left({\frac {y-\mu _{Y} }{\sigma _{Y} } }\right)^{2}\right]\right)$$}@} where $\rho$ is {@{the [correlation](Pearson%20product-moment%20correlation%20coefficient.md) between $X$ and $Y$}@} and where {@{$\sigma _{X}>0$ and $\sigma _{Y}>0$}@}. In this case, {@{$${\boldsymbol {\mu } }={\begin{pmatrix}\mu _{X}\\\mu _{Y}\end{pmatrix} },\quad {\boldsymbol {\Sigma } }={\begin{pmatrix}\sigma _{X}^{2}&\rho \sigma _{X}\sigma _{Y}\\\rho \sigma _{X}\sigma _{Y}&\sigma _{Y}^{2}\end{pmatrix} }.$$}@} (annotation: {@{The precision matrix}@} is {@{$${\boldsymbol {Q} }={\boldsymbol {\Sigma } }^{-1}={\frac {1}{1-\rho ^{2} } }{\begin{pmatrix}{\frac {1}{\sigma _{X}^{2} } }&-{\frac {\rho }{\sigma _{X}\sigma _{Y} } }\\-{\frac {\rho }{\sigma _{X}\sigma _{Y} } }&{\frac {1}{\sigma _{Y}^{2} } }\end{pmatrix} } \,.$$}@}) <!--SR:!2026-04-15,68,337!2026-05-09,87,359!2026-06-15,109,322!2026-05-12,89,362!2026-05-14,91,359!2026-05-04,82,348!2026-05-12,89,362!2026-03-21,45,328-->

In {@{the bivariate case (annotation: but _not_ in the general multivariate case)}@}, {@{the first equivalent condition (annotation: every linear combination of a random vector is normally distributed)}@} for multivariate reconstruction of normality can be made {@{less restrictive}@} as it is sufficient to verify that {@{a [countably infinite](countably%20infinite.md) set of distinct (annotation: i.e. linearly independent) linear combinations of $X$ and $Y$ are normal}@} in order to conclude that {@{the vector of ${\text{[XY]} }\prime$ is bivariate normal}@}.<sup>[\[7\]](#^ref-7)</sup> (annotation: The proof is {@{slightly complicated}@} and involves {@{characteristic functions and Taylor series}@}.) <!--SR:!2026-05-04,82,348!2026-05-01,79,348!2026-05-03,81,348!2026-05-15,92,359!2026-05-06,84,348!2026-05-14,91,359!2026-04-27,76,335-->

#### degenerate case

If {@{the covariance matrix ${\boldsymbol {\Sigma } }$ is not full rank}@}, then the multivariate normal distribution is {@{degenerate and does not have a density}@}. More precisely, it does not have {@{a density with respect to _k_-dimensional [Lebesgue measure](Lebesgue%20measure.md)}@} \(which is {@{the usual measure assumed in calculus-level probability courses}@}\). Only {@{random vectors whose distributions are [absolutely continuous](absolute%20continuity.md#absolute%20continuity%20of%20measures) with respect to a measure}@} are said to have {@{densities \(with respect to that measure\)}@}. To talk about {@{densities but avoid dealing with measure-theoretic complications}@} it can be simpler to restrict {@{attention to a subset of $\operatorname {rank} ({\boldsymbol {\Sigma } })$ of the coordinates of $\mathbf {x}$}@} such that {@{the covariance matrix for this subset is positive definite}@}; then {@{the other coordinates}@} may be thought of as {@{an [affine function](affine%20function.md) of these selected coordinates}@}.<sup>[\[9\]](#^ref-9)</sup> <!--SR:!2026-04-27,76,337!2026-05-15,92,362!2026-04-15,68,337!2026-05-01,81,348!2026-04-17,69,322!2026-05-14,91,362!2026-05-01,81,348!2026-05-07,86,359!2026-05-08,87,359!2026-05-02,80,348!2026-04-22,73,322-->

To talk about {@{densities meaningfully in singular cases}@}, then, we must select {@{a different base measure}@}. Using {@{the [disintegration theorem](disintegration%20theorem.md)}@} we can define {@{a restriction of Lebesgue measure}@} to {@{the $\operatorname {rank} ({\boldsymbol {\Sigma } })$-dimensional affine subspace of $\mathbb {R} ^{k}$ where the Gaussian distribution is supported}@}, i.e. {@{$\left\{ {\boldsymbol {\mu } }+{\boldsymbol {\Sigma ^{1/2} } }\mathbf {v} :\mathbf {v} \in \mathbb {R} ^{k}\right\}$}@}. (annotation: {@{The square root exists}@} due to {@{the [spectral theorem](spectral%20theorem.md)}@}, which guarantees that {@{any symmetric positive semidefinite matrix has a symmetric positive semidefinite square root}@}.) With {@{respect to this measure}@} the distribution has {@{the density of the following motif}@}: {@{$$f(\mathbf {x} )={\frac {\exp \left(-{\frac {1}{2} }\left(\mathbf {x} -{\boldsymbol {\mu } }\right)^{\mathrm {T} }{\boldsymbol {\Sigma } }^{+}\left(\mathbf {x} -{\boldsymbol {\mu } }\right)\right)}{\sqrt {\det \nolimits ^{*}(2\pi {\boldsymbol {\Sigma } })} } }$$}@} where ${\boldsymbol {\Sigma } }^{+}$ is {@{the [generalized inverse](generalized%20inverse.md)}@} (annotation: {@{the [Moore–Penrose inverse](Moore–Penrose%20inverse.md)}@} of {@{the covariance matrix}@}) and $\det \nolimits ^{*}$ is {@{the [pseudo-determinant](pseudo-determinant.md)}@} (annotation: {@{the product of all non-zero eigenvalues}@} of {@{a square matrix}@}).<sup>[\[10\]](#^ref-10)</sup> <!--SR:!2026-05-08,87,362!2026-04-02,56,310!2026-05-06,84,348!2026-04-19,71,337!2026-05-12,91,362!2026-05-03,83,348!2026-05-06,84,348!2026-04-17,67,310!2026-05-02,82,348!2026-05-14,91,359!2026-05-14,91,362!2026-04-30,78,348!2026-05-10,89,362!2026-05-11,90,359!2026-04-28,77,337!2026-04-23,74,322!2026-04-29,80,348!2026-07-13,128,388-->

### cumulative distribution function

### complementary cumulative distribution function \(tail distribution\)

{@{The [complementary cumulative distribution function](cumulative%20distribution%20function.md#derived%20functions) \(ccdf\) or the __tail distribution__}@} is defined as {@{${\overline {F} }(\mathbf {x} )=1-\mathbb {P} \left(\mathbf {X} \leq \mathbf {x} \right)$}@}. ### mutual information

{@{The [mutual information](mutual%20information.md) of two multivariate normal distribution}@} is a special case of {@{the [Kullback–Leibler divergence](Kullback–Leibler%20divergence.md)}@} in which $P$ {@{(annotation: _true_ distribution) is the full $k$ dimensional multivariate distribution}@} and $Q$ {@{(annotation: _approximating_ distribution) is the product of the $k_{1}$ and $k_{2}$ dimensional marginal distributions $X$ and $Y$}@}, such that {@{$k_{1}+k_{2}=k$}@}. {@{The mutual information between $X$ and $Y$}@} is given by:<sup>[\[21\]](#^ref-21)</sup> {@{$$I({\boldsymbol {X} },{\boldsymbol {Y} })={\frac {1}{2} }\ln \left({\frac {\det(\Sigma _{X})\det(\Sigma _{Y})}{\det(\Sigma )} }\right),$$}@} where {@{$$\Sigma ={\begin{bmatrix}\Sigma _{X}&\Sigma _{XY}\\\Sigma _{XY}&\Sigma _{Y}\end{bmatrix} }.$$}@} (annotation: {@{The expression on the right inside $\ln$}@} looks like calculating {@{the reciprocal of the correlation coefficient}@}...) <!--SR:!2026-05-14,91,362!2026-04-29,80,348!2026-05-04,84,348!2026-04-20,67,339!2026-04-26,75,337!2026-05-15,92,362!2026-04-25,76,337!2026-05-14,91,362!2026-05-12,89,359!2026-05-01,79,348-->

If $Q$ is {@{product of $k$ one-dimensional normal distributions}@}, then in the notation of {@{the [Kullback–Leibler divergence section](#Kullback%E2%80%93Leibler%20divergence) of this article}@}, ${\boldsymbol {\Sigma } }_{1}$ {@{(annotation: covariance matrix of _approximating_ distribution) is a [diagonal matrix](diagonal%20matrix.md)}@} with {@{the diagonal entries of ${\boldsymbol {\Sigma } }_{0}$ (annotation: covariance matrix of _true_ distribution)}@}, and {@{${\boldsymbol {\mu } }_{1}={\boldsymbol {\mu } }_{0}$}@}. {@{The resulting formula for mutual information}@} is: {@{$$I({\boldsymbol {X} })=-{1 \over 2}\ln |{\boldsymbol {\rho } }_{0}|,$$}@} where ${\boldsymbol {\rho } }_{0}$ is {@{the [correlation matrix](covariance%20matrix.md#correlation%20matrix) constructed from ${\boldsymbol {\Sigma } }_{0}$}@}.<sup>[\[22\]](#^ref-22)</sup> <!--SR:!2026-05-05,83,348!2026-05-14,91,362!2026-04-23,75,348!2026-05-15,92,362!2026-03-28,52,317!2026-05-06,84,348!2026-03-16,39,322!2026-05-14,91,362-->

In {@{the bivariate case}@} {@{the expression for the mutual information}@} is: {@{$$I(x;y)=-{1 \over 2}\ln(1-\rho ^{2}).$$}@} (annotation: It follows from that {@{the correlation matrix}@} is {@{$\boldsymbol{\rho}_0 = \begin{bmatrix} 1 & \rho \\ \rho & 1 \end{bmatrix}$}@}.) <!--SR:!2026-03-31,55,339!2026-05-14,91,362!2026-05-14,91,362!2026-05-13,92,362!2026-04-29,80,348-->

### joint normality

#### normally distributed and independent

If $X$ and $Y$ are {@{normally distributed and [independent](statistical%20independence.md)}@}, this implies they are {@{"jointly normally distributed"}@}, i.e., {@{the pair $(X,Y)$ must have multivariate normal distribution}@}. However, {@{a pair of jointly normally distributed variables}@} need not {@{be independent \(would only be so if uncorrelated, $\rho =0$\)}@}. <!--SR:!2026-03-31,54,310!2026-05-11,90,359!2026-04-28,77,337!2026-04-25,76,337!2026-04-28,77,335-->

#### two normally distributed random variables need not be jointly bivariate normal

#### correlations and independence

In general, random variables may be {@{uncorrelated but statistically dependent}@}. But, as pointed out just above, it is _not_ true that {@{two random variables}@} that are {@{\(_separately_, marginally\) normally distributed and uncorrelated are independent}@}. <!--SR:!2026-05-11,88,359!2026-04-19,71,337!2026-05-08,87,362!2026-05-08,87,359!2026-05-14,91,359!2026-04-27,76,335!2026-04-28,77,337!2026-04-23,72,322-->

### affine transformation

### geometric interpretation

{@{The equidensity contours of a non-singular}@} multivariate normal distribution are {@{[ellipsoids](ellipsoid.md) \(i.e. affine transformations of [hyperspheres](hypersphere.md)\) centered at the mean}@}.<sup>[\[28\]](#^ref-28)</sup> Hence {@{the multivariate normal distribution}@} is an example of {@{the class of [elliptical distributions](elliptical%20distribution.md)}@}. {@{The directions of the principal axes}@} of the ellipsoids are given by {@{the eigenvectors of the covariance matrix ${\boldsymbol {\Sigma } }$}@}. {@{The squared relative lengths of the principal axes}@} are given by {@{the corresponding eigenvalues}@}. <!--SR:!2026-05-02,82,348!2026-04-28,79,348!2026-04-28,79,348!2026-05-15,92,362!2026-04-30,78,348!2026-05-14,91,362!2026-05-02,82,348!2026-04-13,66,322-->

### parameter estimation

- Further information: ::@:: [Estimation of covariance matrices](estimation%20of%20covariance%20matrices.md) <!--SR:!2026-04-28,77,337!2026-04-27,76,337-->

The derivation of {@{the [maximum-likelihood](maximum%20likelihood.md) [estimator](estimator.md) of the covariance matrix}@} of a multivariate normal distribution is {@{straightforward}@}. <!--SR:!2026-05-14,91,359!2026-05-05,83,348-->

In short, {@{the probability density function \(pdf\) of a multivariate normal}@} is {@{$$f(\mathbf {x} )={\frac {1}{\sqrt {(2\pi )^{k}|{\boldsymbol {\Sigma } }|} } }\exp \left(-{1 \over 2}(\mathbf {x} -{\boldsymbol {\mu } })^{\rm {T} }{\boldsymbol {\Sigma } }^{-1}({\mathbf {x} }-{\boldsymbol {\mu } })\right)$$}@} and {@{the ML estimator of the covariance matrix from a sample of _n_ observations}@} is <sup>[\[31\]](#^ref-31)</sup> {@{$${\widehat {\boldsymbol {\Sigma } } }={1 \over n}\sum _{i=1}^{n}({\mathbf {x} }_{i}-{\overline {\mathbf {x} } })({\mathbf {x} }_{i}-{\overline {\mathbf {x} } })^{\mathrm {T} }$$}@} which is simply {@{the [sample covariance matrix](sample%20covariance%20matrix.md)}@}. This is {@{a [biased estimator](biased%20estimator.md) whose expectation}@} is {@{$$E\left[{\widehat {\boldsymbol {\Sigma } } }\right]={\frac {n-1}{n} }{\boldsymbol {\Sigma } }.$$}@} <!--SR:!2026-05-15,92,362!2026-05-10,89,359!2026-05-01,79,348!2026-04-24,73,322!2026-04-14,67,337!2026-05-10,87,362!2026-03-31,55,339-->

{@{An unbiased sample covariance}@} is {@{${\widehat {\boldsymbol {\Sigma } } }={\frac {1}{n-1} }\sum _{i=1}^{n}(\mathbf {x} _{i}-{\overline {\mathbf {x} } })(\mathbf {x} _{i}-{\overline {\mathbf {x} } })^{\rm {T} }={\frac {1}{n-1} }\left[X'\left(I-{\frac {1}{n} }\cdot J\right)X\right]$}@} \({@{matrix}@} form; $I$ is {@{the $K\times K$ identity matrix, J is a $K\times K$ matrix of ones}@}; {@{the term in parentheses}@} is thus {@{the $K\times K$ centering matrix}@}\) (annotation: {@{The expression inside the summation}@} can be equivalently replaced by {@{$\mathbf x_i (\mathbf x_i - \overline {\mathbf x})^{\rm T}$}@}. Then we see, by stacking {@{the $n$ data points in the correct direction}@}, we get {@{$X' C_n X$}@} where {@{$C_n X$ is $X$ centered by $C_n$ such that each column has a zero mean}@}. This meaning corresponds to the term {@{$(\mathbf x_i - \overline {\mathbf x})^{\rm T}$}@}.) <!--SR:!2026-05-10,88,362!2026-05-10,71,277!2026-05-14,91,362!2026-05-11,90,362!2026-04-12,63,310!2026-05-15,92,362!2026-05-14,91,362!2026-04-17,69,322!2026-05-12,91,362!2026-05-05,83,348!2026-04-20,72,337!2026-04-21,73,335-->

{@{The [Fisher information matrix](Fisher%20information%20matrix.md#matrix%20form)}@} for estimating {@{the parameters of a multivariate normal distribution}@} has {@{a closed form expression}@}. This can be used, for example, to compute {@{the [Cramér–Rao bound](Cramér–Rao%20bound.md) for parameter estimation}@} in this setting. See {@{[Fisher information](Fisher%20information.md#multivariate%20normal%20distribution)}@} for more details. <!--SR:!2026-05-06,84,348!2026-05-10,89,362!2026-05-12,91,359!2026-05-15,92,362!2026-05-02,82,348-->

### Bayesian inference

In {@{[Bayesian statistics](Bayesian%20statistics.md)}@}, {@{the [conjugate prior](conjugate%20prior.md) (annotation: a prior that, when combined with the likelihood, gives a posterior in the same distribution family as the prior)}@} of {@{the mean vector is another multivariate normal distribution}@}, and the conjugate prior of {@{the covariance matrix is an [inverse-Wishart distribution](Inverse-Wishart%20distribution.md) ${\mathcal {W} }^{-1}$}@}. Suppose then that {@{_n_ observations}@} have been made {@{$$\mathbf {X} =\{\mathbf {x} _{1},\dots ,\mathbf {x} _{n}\}\sim {\mathcal {N} }({\boldsymbol {\mu } },{\boldsymbol {\Sigma } })$$}@} and that {@{a conjugate prior}@} has been assigned, where {@{$$p({\boldsymbol {\mu } },{\boldsymbol {\Sigma } })=p({\boldsymbol {\mu } }\mid {\boldsymbol {\Sigma } })\ p({\boldsymbol {\Sigma } }),$$}@} where {@{$$p({\boldsymbol {\mu } }\mid {\boldsymbol {\Sigma } })\sim {\mathcal {N} }({\boldsymbol {\mu } }_{0},m^{-1}{\boldsymbol {\Sigma } }),$$}@} and {@{$$p({\boldsymbol {\Sigma } })\sim {\mathcal {W} }^{-1}({\boldsymbol {\Psi } },n_{0}).$$}@} (annotation: where ___μ₀___ is {@{the prior mean vector}@}, _m_ is {@{the prior precision scaling factor}@}, ___Ψ___ is {@{the scale matrix for the inverse-Wishart prior}@}, and __n₀__ is {@{its degrees of freedom}@}.) Then<sup>[\[31\]](#^ref-31)</sup> {@{$${\begin{array}{rcl}p({\boldsymbol {\mu } }\mid {\boldsymbol {\Sigma } },\mathbf {X} )&\sim &{\mathcal {N} }\left({\frac {n{\bar {\mathbf {x} } }+m{\boldsymbol {\mu } }_{0} }{n+m} },{\frac {1}{n+m} }{\boldsymbol {\Sigma } }\right),\\p({\boldsymbol {\Sigma } }\mid \mathbf {X} )&\sim &{\mathcal {W} }^{-1}\left({\boldsymbol {\Psi } }+n\mathbf {S} +{\frac {nm}{n+m} }({\bar {\mathbf {x} } }-{\boldsymbol {\mu } }_{0})({\bar {\mathbf {x} } }-{\boldsymbol {\mu } }_{0})',n+n_{0}\right),\end{array} }$$}@} (annotation: Update {@{the mean}@} by taking {@{the weighted average of the sample mean and prior mean}@}, with weights proportional to {@{sample size $n$ and prior precision scaling factor $m$}@}; then decrease {@{its uncertainty to $\frac{1}{n+m}\Sigma$}@}. Update {@{the covariance}@} by adding {@{prior scale $\boldsymbol \Psi$, (biased) sample scatter $n \mathbf S$}@}, and an adjustment for {@{the difference between sample mean and prior mean}@}, obtained by taking {@{the outer product of that difference with itself with scaling by $\frac {nm} {n + m}$}@}; then increase {@{degrees of freedom to $n_0 + n$}@}.) where {@{$${\begin{aligned}{\bar {\mathbf {x} } }&={\frac {1}{n} }\sum _{i=1}^{n}\mathbf {x} _{i},\\\mathbf {S} &={\frac {1}{n} }\sum _{i=1}^{n}(\mathbf {x} _{i}-{\bar {\mathbf {x} } })(\mathbf {x} _{i}-{\bar {\mathbf {x} } })'.\end{aligned} }$$}@} <!--SR:!2026-05-15,92,362!2026-04-02,56,310!2026-05-01,79,348!2026-05-06,84,348!2026-04-27,76,337!2026-05-05,83,348!2026-05-10,88,362!2026-04-19,71,335!2026-04-03,56,342!2026-04-21,73,337!2026-05-07,86,359!2026-05-06,84,348!2026-03-21,45,328!2026-04-02,54,310!2026-03-26,31,248!2026-05-01,79,348!2026-05-03,81,348!2026-04-29,80,348!2026-05-09,88,362!2026-05-05,83,348!2026-04-22,71,322!2026-05-02,82,348!2026-05-03,83,348!2026-05-10,87,359!2026-05-06,84,348-->

### multivariate normality tests

{@{Multivariate normality tests}@} check {@{a given set of data}@} for similarity to {@{the multivariate [normal distribution](normal%20distribution.md)}@}. {@{The [null hypothesis](null%20hypothesis.md)}@} is that {@{the [data set](data%20set.md) is similar to the normal distribution}@}, therefore {@{a sufficiently small [_p_-value](p-value.md)}@} indicates {@{non-normal data}@}. {@{Multivariate normality tests}@} include {@{the Cox–Small test<sup>[\[32\]](#^ref-32)</sup> and Smith and Jain's adaptation}@}<sup>[\[33\]](#^ref-33)</sup> of {@{the Friedman–Rafsky test created by [Larry Rafsky](Larry%20Rafsky.md) and [Jerome Friedman](Jerome%20H.%20Friedman.md)}@}.<sup>[\[34\]](#^ref-34)</sup> <!--SR:!2026-05-09,88,362!2026-04-29,80,348!2026-05-10,87,362!2026-05-03,81,348!2026-04-23,72,322!2026-05-03,83,348!2026-05-15,92,362!2026-05-13,90,362!2026-04-24,73,322!2026-05-05,83,348-->

#### Mardia's test

{@{__Mardia's test__}@}<sup>[\[35\]](#^ref-35)</sup> is based on {@{multivariate extensions of [skewness](skewness.md) and [kurtosis](kurtosis.md) measures}@}. For {@{a sample {__x__<sub>1</sub>, ..., __x__<sub>_n_</sub>} of _k_-dimensional vectors}@} we compute {@{$${\begin{aligned}&{\widehat {\boldsymbol {\Sigma } } }={1 \over n}\sum _{j=1}^{n}\left(\mathbf {x} _{j}-{\bar {\mathbf {x} } }\right)\left(\mathbf {x} _{j}-{\bar {\mathbf {x} } }\right)^{\mathrm {T} }\\&A={1 \over 6n}\sum _{i=1}^{n}\sum _{j=1}^{n}\left[(\mathbf {x} _{i}-{\bar {\mathbf {x} } })^{\mathrm {T} }\;{\widehat {\boldsymbol {\Sigma } } }^{-1}(\mathbf {x} _{j}-{\bar {\mathbf {x} } })\right]^{3}\\&B={\sqrt {\frac {n}{8k(k+2)} } }\left\{ {1 \over n}\sum _{i=1}^{n}\left[(\mathbf {x} _{i}-{\bar {\mathbf {x} } })^{\mathrm {T} }\;{\widehat {\boldsymbol {\Sigma } } }^{-1}(\mathbf {x} _{i}-{\bar {\mathbf {x} } })\right]^{2}-k(k+2)\right\}\end{aligned} }$$}@} (annotation: _A_ is {@{a multivariate extension of _skewness_}@}, noting the {@{_third_ power of the _squared_ Mahalanobis distance}@} is taken, and the "_squared_" part is somewhat explained by that {@{_A_ approximately follows a chi-squared distribution}@}. _B_ is {@{a multivariate extension of _kurtosis_}@}, noting the {@{_second_ power of the _squared_ Mahalanobis distance}@} is taken, analogous to taking {@{the _fourth_ power of the standard score}@}. That _B_ follows {@{a standard normal distribution instead of a nonnegative distribution}@} is somewhat explained by {@{the subtraction}@}.) Under {@{the null hypothesis of multivariate normality, the statistic}@} _A_ will have {@{approximately a [chi-squared distribution](chi-squared%20distribution.md) with ⁠1/6⁠⋅<!-- markdown separator -->_k_\(_k_ + 1\)\(_k_ + 2\) degrees of freedom}@}, and _B_ will be {@{approximately [standard normal](standard%20normal.md#standard%20normal%20distribution) _N_\(0,1\)}@}. <!--SR:!2026-03-31,55,339!2026-05-14,91,359!2026-05-12,89,362!2026-03-18,42,308!2026-05-01,79,348!2026-05-05,83,348!2026-05-06,84,348!2026-04-27,76,335!2026-04-11,59,328!2026-04-26,75,337!2026-04-26,77,337!2026-04-25,74,322!2026-04-16,69,337!2026-05-11,90,359!2026-04-26,77,337-->

{@{Mardia's kurtosis statistic (annotation: _B_)}@} is {@{skewed and converges very slowly}@} to {@{the limiting normal distribution}@}. For {@{medium size samples $(50\leq n<400)$}@}, {@{the parameters of the asymptotic distribution}@} of the kurtosis statistic are {@{modified}@}<sup>[\[36\]](#^ref-36)</sup> For {@{small sample tests \($n<50$\)}@} {@{empirical critical values}@} are used. {@{Tables of critical values for both statistics}@} are given by {@{Rencher<sup>[\[37\]](#^ref-37)</sup> for _k_ = 2, 3, 4}@}. <!--SR:!2026-04-28,77,337!2026-05-05,83,348!2026-04-28,79,348!2026-05-14,91,362!2026-05-06,84,348!2026-05-12,89,359!2026-04-15,68,335!2026-05-13,92,362!2026-05-04,84,348!2026-05-11,88,362-->

{@{Mardia's tests}@} are {@{affine invariant but not consistent}@}. For example, {@{the multivariate skewness test}@} is not {@{consistent against symmetric non-normal alternatives}@}.<sup>[\[38\]](#^ref-38)</sup> <!--SR:!2026-05-09,87,362!2026-04-29,80,348!2026-05-05,83,348!2026-05-09,88,362-->

#### BHEP test

{@{The __BHEP test__}@}<sup>[\[39\]](#^ref-39)</sup> computes {@{the norm of the difference}@} between {@{the empirical [characteristic function](characteristic%20function%20(probability%20theory).md) and the theoretical characteristic function of the normal distribution}@}. {@{Calculation of the norm}@} is performed in {@{the [L<sup>2</sup>\(_μ_\)](Lp%20space.md) space of square-integrable functions}@} with respect to {@{the Gaussian weighting function $\mu _{\beta }(\mathbf {t} )=(2\pi \beta ^{2})^{-k/2}e^{-|\mathbf {t} |^{2}/(2\beta ^{2})}$}@}. {@{The test statistic}@} is {@{$${\begin{aligned}T_{\beta }&=\int _{\mathbb {R} ^{k} }\left|{1 \over n}\sum _{j=1}^{n}e^{i\mathbf {t} ^{\mathrm {T} }{\widehat {\boldsymbol {\Sigma } } }^{-1/2}(\mathbf {x} _{j}-{\bar {\mathbf {x} )} } }-e^{-|\mathbf {t} |^{2}/2}\right|^{2}\;{\boldsymbol {\mu } }_{\beta }(\mathbf {t} )\,d\mathbf {t} \\&={1 \over n^{2} }\sum _{i,j=1}^{n}e^{-{\beta ^{2} \over 2}(\mathbf {x} _{i}-\mathbf {x} _{j})^{\mathrm {T} }{\widehat {\boldsymbol {\Sigma } } }^{-1}(\mathbf {x} _{i}-\mathbf {x} _{j})}-{\frac {2}{n(1+\beta ^{2})^{k/2} } }\sum _{i=1}^{n}e^{-{\frac {\beta ^{2} }{2(1+\beta ^{2})} }(\mathbf {x} _{i}-{\bar {\mathbf {x} } })^{\mathrm {T} }{\widehat {\boldsymbol {\Sigma } } }^{-1}(\mathbf {x} _{i}-{\bar {\mathbf {x} } })}+{\frac {1}{(1+2\beta ^{2})^{k/2} } }\end{aligned} }$$}@} (annotation: {@{The empirical characteristic function}@} approximates {@{the theoretical one $\varphi(\mathbf{t}) = \mathbb{E}\ [e^{i \mathbf{t}^T \mathbf{X} }]$}@} by replacing {@{the expectation with the sample mean}@}: {@{$\hat{\varphi}(\mathbf{t}) = \frac{1}{n} \sum_{j=1}^n e^{i \mathbf{t}^T \mathbf{x}_j}$}@}. For {@{the standard multivariate normal $N_k(\mathbf{0}, \mathbf{I}_k)$}@}, {@{the theoretical characteristic function}@} is {@{$\varphi(\mathbf{t}) = e^{-\frac{|\mathbf{t}|^2}{2} }$}@}. In {@{the BHEP test}@}, data are {@{_centered_ $(\mathbf{x}_j - \bar{\mathbf{x} })$ and _standardized_ using $\hat{\boldsymbol{\Sigma} }^{-1/2}$}@}, meaning each observation is shifted {@{to zero mean}@} and scaled {@{by the inverse square root of the sample covariance matrix}@} to match {@{the null hypothesis of $N_k(\mathbf{0}, \mathbf{I}_k)$}@}. {@{The Gaussian weighting function $\left(2 \pi \beta^2\right)^{-k/2} e^{- \lvert \mathbf t \rvert^2 / 2 \beta^2}$}@} ensures {@{the integral is finite}@} and emphasizes {@{differences near the origin by down-weighting large $\lvert\mathbf{t}\rvert$}@}, where {@{characteristic functions decay slowly}@}.) {@{The limiting distribution of this test statistic}@} is {@{a weighted sum of chi-squared random variables}@}.<sup>[\[39\]](#^ref-39)</sup> <!--SR:!2026-05-06,84,348!2026-05-15,92,359!2026-05-02,82,348!2026-05-03,83,348!2026-05-09,86,359!2026-05-11,88,359!2026-05-14,91,362!2026-06-04,91,302!2026-05-08,86,362!2026-05-14,91,362!2026-04-26,75,337!2026-04-28,77,337!2026-05-10,89,359!2026-03-22,46,339!2026-05-12,91,362!2026-05-12,91,362!2026-05-12,89,362!2026-05-02,82,348!2026-04-19,68,342!2026-03-31,54,310!2026-03-20,44,302!2026-05-15,92,359!2026-05-05,83,348!2026-05-11,88,359!2026-05-08,87,362!2026-04-15,68,335-->

## computational methods

### drawing values from the distribution

{@{A widely used method}@} for {@{drawing \(sampling\) a random vector __x__}@} from {@{the _N_-dimensional multivariate normal distribution with mean vector __μ__ and [covariance matrix](covariance%20matrix.md) __Σ__}@} works as follows:<sup>[\[41\]](#^ref-41)</sup> (annotation: 3 steps: {@{find the square root of the covariance matrix, generate standard normal random vector, affine transform}@}) <!--SR:!2026-05-06,84,348!2026-05-04,82,348!2026-05-08,87,359!2026-05-11,90,362-->

1. Find {@{any real matrix __A__}@} such that {@{__AA__<sup>T</sup> = __Σ__}@}. When {@{__Σ__ is positive-definite}@}, {@{the [Cholesky decomposition](Cholesky%20decomposition.md)}@} is typically used because it is {@{widely available, computationally efficient, and well known}@}. If {@{a rank-revealing \(pivoted\) Cholesky decomposition such as LAPACK's dpstrf\(\)}@} is available, it can be used in {@{the general positive-semidefinite case}@} as well. {@{A slower general alternative}@} is to use {@{the matrix __A__ = __UΛ__<sup>1/2</sup>}@} obtained from {@{a [spectral decomposition](eigendecomposition%20of%20a%20matrix.md#real%20symmetric%20matrices) __Σ__ = __UΛU__<sup>−1</sup> of __Σ__}@}.
2. Let {@{__z__ = \(_z_<sub>1</sub>, ..., _z<sub>N</sub>_\)<sup>T</sup>}@} be {@{a vector whose components are _N_ [independent](statistical%20independence.md) [standard normal](normal%20distribution.md) variates}@} \(which can be generated, for example, by using {@{the [Box–Muller transform](Box–Muller%20transform.md)}@}\).
3. Let {@{__x__ be __μ__ + __Az__}@}. This has {@{the desired distribution}@} due to {@{the affine transformation property}@}. <!--SR:!2026-05-12,89,359!2026-05-02,82,348!2026-04-28,77,337!2026-05-02,80,348!2026-05-04,82,348!2026-05-14,91,359!2026-05-08,87,362!2026-04-18,70,322!2026-04-26,77,337!2026-05-03,81,348!2026-05-06,84,348!2026-05-15,92,362!2026-05-15,92,359!2026-05-04,84,348!2026-05-03,81,348!2026-04-16,69,337-->
