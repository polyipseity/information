# properties of metals, metalloids, and nonmetals

## comparison

Compare with [nonmetal § properties](nonmetal.md#properties).

| property                                                | metal                                                                                                                                                                                                                                                                                                                                                              | metalloid                                            | nonmetal                                                                                                                                                              |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| appearance                                              | - mostly shiny and grey-white                                                                                                                                                                                                                                                                                                                                      | - shiny and grey-white                               | - mostly dull                                                                                                                                                         |
| deformability (as a solid)                              | - mostly [ductile](ductility.md) and [malleable](malleability.md) <br/> - some brittle                                                                                                                                                                                                                                                                             | - brittle                                            | - brittle <br/> - some non-brittle                                                                                                                                    |
| [density](density.md)                                   | - mostly high <br/> - exceptional example(s): [alkali metals](alkali%20metal.md)                                                                                                                                                                                                                                                                                   | - between metals and nonmetals                       | - mostly low                                                                                                                                                          |
| [electrical conductivity](electrical%20conductivity.md) | - good to high                                                                                                                                                                                                                                                                                                                                                     | - intermediate to good                               | - poor to good                                                                                                                                                        |
| form                                                    | - almost all [solid](solid.md) <br/> - [liquid](liquid.md): [Cs](caesium.md) (near [room temperature](room%20tempeature.md)), [Fr](francium.md) (near [room temperature](room%20tempeature.md)), [Ga](gallium.md) (near [room temperature](room%20tempeature.md)), [Hg](mercury%20(element).md), [Rb](rubidium.md) (near [room temperature](room%20tempeature.md)) | - all [solid](solid.md)                              | - mostly [gas](gas.md) <br/> - [solid](solid.md): [C](carbon.md), [I](iodine.md), [P](phosphorus.md), [Se](selenium.md) <br/> - [liquid](liquid.md): [Br](bromine.md) |
| [melting point](melting%20point.md)                     | - mostly high <br/> - exceptional example(s): [Na](sodium.md)                                                                                                                                                                                                                                                                                                      | - mostly high                                        | - mostly low                                                                                                                                                          |
| [thermal conductivity](thermal%20conductivity.md)       | - medium to high                                                                                                                                                                                                                                                                                                                                                   | - mostly intermediate <br/> - high: [Si](silicon.md) | - almost negligible to very high                                                                                                                                      |
