# F-score

## definition

The traditional {{F-measure or balanced F-score (__F<sub>1</sub> score__)}} is {{the [harmonic mean](harmonic%20mean.md#two%20numbers) of [precision and recall](precision%20and%20recall.md)}}: {{$$F_1 = \left( \frac {\text{recall}^{-1} + \text{precision}^{-1} } 2 \right)^{-1} = \frac 2 {\text{recall}^{-1} + \text{precision}^{-1} } = 2 \frac {\text{recall} \cdot \text{precision} } {\text{recall} + \text{precision} } = \frac {2\text{TP} } {2\text{TP} + \text{FP} + \text{FN} }$$}}. <!--SR:!2024-12-05,115,290!2024-09-20,67,270!2024-08-24,44,250-->
