# cluster analysis

## definitions

Typical cluster models include:

- connectivity models ::: For example, the [hierarchical clustering](hierarchical%20clustering.md) builds or divides into clusters based on distance connectivity. <!--SR:!2024-09-08,105,290!2024-07-08,67,310-->
- centroid models ::: For example, the [k-means clustering](k-means%20clustering.md) represents each cluster by its mean vector. <!--SR:!2024-12-27,192,310!2024-06-28,59,310-->

## applications

Clustering are used in {{many fields}}, and can be used for {{compressing, summarizing, or understanding}} data. <!--SR:!2024-07-09,68,310!2024-10-12,122,270-->
