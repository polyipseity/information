# cluster analysis

## definitions

Typical cluster models include:

- connectivity models ::: For example, the [hierarchical clustering](hierarchical%20clustering.md) builds or divides into clusters based on distance connectivity. <!--SR:!2024-04-15,4,270!2024-04-15,4,270-->
- centroid models ::: For example, the [k-means clustering](k-means%20clustering.md) represents each cluster by its mean vector. <!--SR:!2024-04-15,4,270!2024-04-15,4,270-->

## applications

Clustering are used in {{many fields}}, and can be used for {{compressing, summarizing, or understanding}} data. <!--SR:!2024-04-15,4,270!2024-04-15,4,270-->
