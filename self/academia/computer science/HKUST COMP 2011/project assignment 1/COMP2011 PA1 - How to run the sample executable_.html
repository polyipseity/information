How to run pa1.exe?

Step 1: Connect to the ITSC Virtual Barn

Use VMware Horizon Client and Choose Programming Software (this VM is usually faster)

Step 2: Locate the P (Personal Drive) in Virtual Barn

Step 3: Download pa1.exe and the input testcases to the P drive

Download pa1.exe (Some web browser may warn you, you need to accept the warning and download it anyway) and copy it to the P drive

In virtual barn, the default download folder is P:\Downloads. We assume the files are stored in this folder.

You need to create a new empty folder (e.g., pa1) inside the P drive

After that, you need to copy pa1.exe and the testcases to the empty folder

Here is the location of pa1.exe, and there is a sub-folder storing the testcases:

Here is the location of the input testcases:

You can prepare your own testcases and put them in this folder

Step 4: Open Powershell in Windows

You can search Powershell from the start menu

Step 5: Change to the correct directory in Powershell

You need to change to the correct directory

Run P: to change to the P drive

Run cd Downloads to change to the Downloads folder inside the P drive

Run cd pa1 to change to the pa1 folder

Run ls command to check the content of the current directory

Step 6: Run the sample program

A sample command: Get-Content testcase/input1.txt | ./pa1 > myOutput1.txt to run the first test case

You can write your own test case, put it inside the testcase folder, and use a similar command to run your own test case

You can use vscode to open the output file (e.g., Code myOutput1.txt), or drag and drop the text file to any editor.
