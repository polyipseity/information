# lens

## imaging properties

Images created by a lens can be described as {{[real](real%20image.md), [virtual](virtual%20image.md), magnified, diminished, erect, or inverted}}. Images created by a [mirror](mirror.md) are described as {{same size and laterally inverted}}. <!--SR:!2023-05-30,16,270!2023-07-20,53,290-->
