How to run pa2.exe?

Step 1: Connect to the ITSC Virtual Barn

Use VMware Horizon Client and Choose Programming Software (this VM is usually faster)

Step 2: Locate the P (Personal Drive) in Virtual Barn

Step 3: Download pa2.exe and the input testcases to the P drive

Download pa2.exe (Some web browser may warn you, you need to accept the warning and download it anyway) and copy it to the P drive

In virtual barn, the default download folder is P:\Downloads. We assume the files are stored in this folder.

You need to create a new empty folder (e.g., pa2) inside the P drive

After that, you can run pa2.exe.
