Lab 5 Implementing a Zoo Management System


Introduction

In this lab, you will implement a zoo manmagement system that can handle different types of animals, i.e., carnivores and herbivores. Also, this system can calculate the total food cost for all animals in the zoo.


Class Description

Animal serves as a base class for general animal classes.
Derived from it, we have two types of animals:

  1. Carnivore: only eat meat.
  2. Herbivore: only eat vegetables.
The Zoo System can record the animals and calculate the total food consumption.

The class description of Animal and its derived classes are as follows:

  1. Animal:
    • Data Members:
      • name: Stores the name of animal.
      • age: Stores the name of animal.
      • foodPerDay: Stores the daily food consumption of the animal (in kilograms).
      • foodType: Stores the type of food the animal eats (meat or vegetables). FoodType is an enum defined in FoodType.h. This enum is expected to have values like meat and vegetables.
    • Member Functions:
      • Animal(const string& name, int age, double foodPerDay, FoodType foodType): Constructor that initializes the data members.
      • virtual void describe() const = 0: Pure virtual function that requires derived classes to provide an implementation.
      • virtual double getFoodPerDay() const: Return the daily food consumption of the animal.
      • FoodType getFoodType() const: Return the type of food the animal eats.
      • virtual ~Animal(): Virtual destructor.
  2. Carnivore: Derived from Animal
    • Data Members:
      • teethCount: The number of teeth of the carnivore.
    • Member Functions:
      • Carnivore(const string& name, int age, double foodPerDay);: Constructor that initialized a Carnivore object.
      • void describe() const override: Override the pure virtual function from Animal to provide a description specific to carnivores.
  3. Herbivore: Derived from Animal
    • Data Members:
      • favoritePlant: The favorite plant of the herivores.
      • habitatPerference: The habitat preference of the herbivores.
    • Member Functions:
      • Herbivore(const string& name, int age, double foodPerDay): Constructor that initialized a Herbivore object.
      • void describe() const override: Override the pure virtual function from Animal to provide a description specific to carnivores.
  4. ZooSystem:
    • Data Members:
      • Animal** animals: Dynamic array of pointers to Animal.
      • int capacity: The capacity of the dynamic array.
      • int size: The number of animals in the zoo system.
    • Member Functions:
      • ZooSystem(int cap): Initialize the zoo system and also allocating the dynamic array with capacity of cap appropriately.
      • ~ZooSystem(): Deallocate all the animals pointed and the dynamic array, animal.
      • void addAnimal(Animal* animal): Add an animal to the zoo system by appending the address of it to the dynamic array, animals. Do nothing if the array is full.
      • void listAllAnimals() const: List all animals in the zoo system.
      • void listCarnivores() const;: List all carnivores animals in the zoo system.
      • void listHerbivores() const;: List all herbivores animals in the zoo system.
      • double calculateMeatConsumption() const: Calculate the total food consumption of all carnivores in the zoo system.
      • double calculateVegetableConsumption() const: Calculate the total food consumption of all herbivores in the zoo system.

Working Example


Here is a working example of zoo management system. (This main.cpp is provided in the skeleton code) 
#include "ZooSystem.h"
#include "SpecificAnimal.h"
#include "Animal.h"
#include <iostream>

using namespace std;
int main() {
  ZooSystem zoo(50);
  
  zoo.addAnimal(new Lion("Simba", 5, 5.0, 20));
  zoo.addAnimal(new Elephant("Dumbo", 10, 50.0, "Grass", "Savanna"));
 
  // list all animals in the zoo
  cout << "All animals in the zoo:" << endl;
  zoo.listAllAnimals();

  // list all carnivores in the zoo
  cout << "Carnivores in the zoo:" << endl;
  zoo.listCarnivores();

  // list all herbivores in the zoo
  cout << "Herbivores in the zoo:" << endl;
  zoo.listHerbivores();

  // calculate total food consumption per day
  cout << "Total meat consumption per day: " << zoo.calculateMeatConsumption() << " kg" << endl;
  cout << "Total vegetable consumption per day: " << zoo.calculateVegetableConsumption() << " kg" << endl;

  return 0;
}

  • Todo 2.1: Implement the Carnivore class with the given data members and member functions.
  • Todo 2.2: Implement the Herbivore class with the given data members and member functions.
Task 3: Implement the Specific Animal Classes class.
  • Todo 3.1: Implement the specific animal class for carnivores, including Lion, and Tiger class, inheriting from the corresponding dietary class, and override the describe method. (Please refer to Work Example)

Task 5: Test your implementation.
  • Test your implementation with the provided main.cpp and its expected output.

Implementation Details

There are some important points to note when implementing the classes:
  • The Animal class is an abstract base class, and the describe method is a pure virtual function. You need to implement the describe method in the derived classes.
  • The Animal class has a virtual destructor. You need to implement the destructor in the derived classes.
  • The SpecificAnimal class is a derived class of the Animal class. You need to implement the describe method in the derived classes.
  • For the order of addition, the addAnimal method in the ZooSystem class should add the animal to the end of the animals vector.
  • The listAllAnimals method in the ZooSystem class should list all animals in the animals vector, according to the addition order.
  • The listCarnivores method in the ZooSystem class should list all carnivores in the animals vector, according to the addition order of carnivores. (The same as the herbivores.)

Basic Code Usages

A Makefile is provided to you in the skeleton code: type make compile your code.

Alternatively, you can use the following command:

g++ -std=c++11 Animal.cpp SpecificAnimal.cpp ZooSystem.cpp main.cpp -o lab5.exe

For debuging and memory leakage check, use:

g++ -std=c++11 -g -fsanitize=address,leak,undefined Animal.cpp SpecificAnimal.cpp ZooSystem.cpp main.cpp -o lab5.exe

And then run the program with:

./lab5.exe
