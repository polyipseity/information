Lab 1 Class Basics

Overview

In this lab, we will simulate a gradebook storing a list of students in a class. The gradebook should store a student's full information including name, student ID, department/major, year of study, and of course the student's scores. This is accomplished by implementing a few classes and structs with different attributes and relationships.

The following struct and class are given in student.h and the member functions will need to be implemented in student.cpp: 

struct CourseGrade {
  double labs;
  double assignments;
  double midtermExam;
  double finalExam;

  double getTotalPercentage() const;
};

class Student {
  private:
      string name;
      int SID;
      string department;
      int year;
      CourseGrade grade;

  public:
      Student();
      Student(const string& name, int SID, const string& department, int year);

      void setGrade(const CourseGrade& grade);

      string getName() const; 
      int getSID() const;
      string getDepartment() const;
      int getYear() const;
      double getTotalPercentage() const;

      void display() const {
        // given
      }
};

The struct CourseGrade contains the grade composition of a student. You will need to implement a member function to calculate the total course grade using the weights defined as constants. Note that in C++, structs are essentially similar to classes (except the default visibility of attributes and methods are public).

The class Student represents a student with name, student ID, department, year of study and grades. You need to finish the implementation of the non-default constructor and the accessor/mutator functions.

The following class is defined in studentlist.h and the member functions will need to be implemented in studentlist.cpp: 

const int CLASS_CAPACITY = 100;
const string ALL_DEPARTMENTS = "ALL";
const int ALL_YEARS = -1;

class StudentList {
    private:
        Student students[CLASS_CAPACITY];
        int numStudents {0};

    public:
        StudentList() = default;

        void addStudent(const string& name, int SID, const string& department, int year);
        void setStudentGrade(int SID, const CourseGrade& grade);

        void displaySortedName(const string& depFilter = ALL_DEPARTMENTS, int yearFilter = ALL_YEARS) const;
        void displaySortedGrade(const string& depFilter = ALL_DEPARTMENTS, int yearFilter = ALL_YEARS) const;
};

The class StudentList contains an array of Students. It supports adding a student, setting the score of a student and display the list of students in sorted order, with possible filters for students of a specific department or year of study. For simplicity, you can assume no 2 students with the same ID or name will be added to the list.

Lab Tasks

Your task is to implement the functions in student.cpp (TODO 1 ~ TODO 4) and studentlist.cpp (TODO 5 ~ TODO 7).

TODO 1: CourseGrade::getTotalPercentage()
Calculate the total course grade using the struct's data members along with the constants defined in student.h representing the grading assignment's course percentage. For example, if a student scores 0.6 in labs, 0.7 in assignments, 0.8 in midterm and 0.9 in final, the total percentage is:

0.6 * LAB_PERCENTAGE + 0.7 * ASGN_PERCENTAGE + 0.8 * MIDTERM_PERCENTAGE + 0.9 * FINAL_PERCENTAGE = 79.6

TODO 2: Student non-default constructor
Initialize the data members with the specified parameters, while grade is initialized to 0 similar to default constructor.
TODO 3: Student mutator function
Implement Student::setGrade() to set the data member accordingly.
TODO 4: Student accessor functions
Implement the 5 accessor functions to return the data member accordingly.
TODO 5: StudentList::addStudent(string name, int SID, string department, int year)
Set the Student in the array at the index indicated by numStudents to the provided parameters, and increment the numStudents counter. If the array is already full, the function should do nothing.
TODO 6: void StudentList::setStudentGrade(int SID, const CourseGrade& grade)
Find the student whose SID is the same as the provided SID, and set that student's score to the provided parameter.
TODO 7: StudentList::displaySortedGrade()
Display the student list in descending grade order. Most of the function has been done for you, which is the same as the function sorting students by name. The function creates a temporary array copying the original array, then sorts it using swap sort. You only need to change the swap condition such that the order is descending total course grade. Note: For simplicity, you may assume no 2 students have the same total grade.

Compile and Test

After you have finished all tasks (TODO 1 ~ TODO 7), you can compile the files by typing g++ *.cpp -Wall -std=c++11 -o lab1.exe in the terminal to produce the executable lab1.exe. Your program must be able to re-generate exactly the same output as the sample output that has been shown in the overview section. You should read the main.cpp file carefully to understand how the sample output is generated.
