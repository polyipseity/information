Lab 8 Shopping Cart Manager


Introduction

The Shopping Cart Manager is a C++ application that leverages the power of the Standard Template Library (STL) to provide users with a comprehensive set of tools for managing their shopping carts. By utilizing STL's containers, algorithms, and iterators, the system delivers a highly efficient and versatile shopping experience. At the core of the Shopping Cart Manager is the use of STL vectors to store and manipulate the products in the user's shopping cart. The application employs STL iterators and algorithms to seamlessly traverse the vector of products, enabling a range of operations. The system includes basic functions such as adding/deleting items, calculating total price, sorting, and filtering, as well as some advanced functions, such as applying discounts and filtering by price range.


Class Description

The Product represents an individual product in the shopping cart. It encapsulates the product's name and price as member variables. It provides a constructor to initialize these properties. It includes two static member functions, compareByPrice and isExpensive, which enable sorting and filtering of products based on price. It also provides print() to print the product's name and price.

The ShoppingCart is the main component of the system, responsible for managing the user's shopping cart. It uses a std::vector container from the STL to store the Product objects in the cart. The class provides several member functions to interact with the shopping cart: 

addProduct: Adds a new product to the cart.
removeProduct: Removes a product from the cart by its name.
getTotalPrice: Calculates the total price of all products in the cart.
sortByPrice: Sorts the products in the cart based on their price.
sortByName: Sorts the products in the cart based on their name.
applyDiscount: Applies a discount to all products in the cart.
getExpensiveProducts: Retrieves a vector of products that exceed a specified price threshold.
getProductsInPriceRange: Retrieves a vector of products in a specified price range.

The displayProducts member function is provided to display the current contents of the shopping cart, which is implemented in the code provided.

std::vector ShoppingCart::getExpensiveProducts(double threshold) const {
  std::vector expensiveProducts;

  std::copy_if(products.begin(), products.end(), std::back_inserter(expensiveProducts),
              [&threshold](const Product& p) { return Product::isExpensive(p, threshold); });

  std::sort(expensiveProducts.begin(), expensiveProducts.end(), Product::compareByPrice);
  return expensiveProducts;
}
