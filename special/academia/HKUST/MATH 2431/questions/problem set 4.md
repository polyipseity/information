> Benford's law and sigma-algebras. Let $M_t=\bigcup_{n\in\mathbb Z}10^n[1,t)$ for $1<t<10$, let $\mathcal M=\sigma(\{M_t:1<t<10\})$, and define $P(M_t)=\log_{10}(t)$.
>
> - (a) Show that $\alpha M_t\in\mathcal M$ for all $\alpha>0$.
> - (b) Show that $P(\alpha M_t)=P(M_t)$.
> - (c) Deduce that the first-digit distribution is Benford's law, i.e. $P(\text{first digit}=k)=\log_{10}(1+1/k)$.
> - (d) Extend the scaling properties from the generating family to all of $\mathcal M$.
>
> Solution:
>
> - (a) The key idea is that {@{multiplying by $10$ does not change the shape of these sets}@}, because the definition {@{already contains all decimal scales $10^n$}@}. So it is enough to study {@{$\alpha\in(1,10)$}@}. If {@{$\alpha t<10$, then multiplying shifts $[1,t)$ to $[\alpha,\alpha t)$}@}, which is {@{exactly $M_{\alpha t}\setminus M_{\alpha}$}@}. If {@{$\alpha t>10$, the scaled interval crosses the digit boundary at $10$}@}, so split {@{it there and rewrite each piece in the same generator form}@}. In either case {@{the result is $\alpha M_t\in\mathcal M$}@}.
> - (b) In the first case, {@{the logarithms cancel as $P(\alpha M_t)=\log_{10}(\alpha t)-\log_{10}(\alpha)=\log_{10}(t)$}@}; in the second case, the same logarithmic cancellation still gives {@{the same value $P(\alpha M_t)=\log_{10}(t)$}@}.
> - (c) A number has first digit ${k}$ {@{exactly when it lies in one of the intervals $10^n[k,k+1)$}@}. So the event is {@{the union $\bigcup_{n\in\mathbb Z}10^n[k,k+1)$, i.e. the slice between $M_k$ and $M_{k+1}$}@}. Therefore its {@{probability is $P(\text{first digit}=k)=P(M_{k+1})-P(M_k)=\log_{10}(k+1)-\log_{10}(k)=\log_{10}(1+1/k)$}@}.
> - (d) We now extend {@{the two scaling properties from the generators to all of $\mathcal M$}@} more concisely. Fix {@{$\alpha>0$ and define $\mathcal M' =\{A\in\mathcal M: \alpha A\in\mathcal M\}$}@}. Because {@{multiplication by $\alpha$ is a bijection of $(0,\infty)$}@}, it preserves {@{the whole space, complements, and countable unions}@}, so {@{$\mathcal M'$ is a sigma-algebra}@}. Part (a) shows {@{every generator $M_t$ lies in $\mathcal M'$}@}, hence by minimality of {@{$\mathcal M=\sigma(\{M_t:1<t<10\})$, we have $\mathcal M\subseteq\mathcal M'\subseteq\mathcal M$, so $\mathcal M'=\mathcal M$}@}. Thus for every {@{$A\in\mathcal M$, the scaled set $\alpha A$ still belongs to $\mathcal M$}@}. <br/><br/> Next let {@{$\mathcal M''=\{A\in\mathcal M:P(\alpha A)=P(A)\}$}@}. This is {@{a Dynkin system}@}: it {@{contains $(0,\infty)$}@}, is closed under {@{complements because $\alpha(A^c)=(\alpha A)^c$ and $P((0,\infty))=1$}@}, and is closed under {@{pairwise disjoint countable unions because multiplication by $\alpha$ preserves disjointness and $P$ is countably additive}@}. By part (b), {@{every generator $M_t$ lies in $\mathcal M''$}@}. Also {@{$\{M_t:1<t<10\}$ is a $\pi$-system because $M_s\cap M_t=M_{\min\{s,t\}}$}@}. Therefore {@{the $\pi$-$\lambda$ theorem gives $\mathcal M=\sigma(\{M_t:1<t<10\})\subseteq\mathcal M''$}@}. Since {@{$\mathcal M''\subseteq\mathcal M$ by definition, we get $\mathcal M''=\mathcal M$}@}. Hence for every {@{$A\in\mathcal M$, both $\alpha A\in\mathcal M$ and $P(\alpha A)=P(A)$}@}.

> More on sigma-algebras.
>
> - (a) Give an example showing that the union of two sigma-algebras on the same space need not be a sigma-algebra.
> - (b) Starting from the definition $\mathcal B(\mathbb R)=\sigma(\{[a,b):a<b\})$, show that $[a,b]\in\mathcal B(\mathbb R)$ and $(-\infty,a]\in\mathcal B(\mathbb R)$.
> - (c) Show that $\mathcal B(\mathbb R)$ is also generated by the half-lines $(-\infty,b]$ and by the collection of all open sets.
>
> Solution:
>
> - (a) Take {@{$\Omega=\{1,2,3,4\}$}@} with {@{$\mathcal F_1=\{\varnothing,\{1\},\{2,3,4\},\Omega\}$ and $\mathcal F_2=\{\varnothing,\{2\},\{1,3,4\},\Omega\}$}@}. Their union contains {@{$\{1\}$ and $\{2\}$}@} but not {@{$\{1,2\}$}@}, so it fails closure under unions and is {@{not a sigma-algebra}@}.
> - (b) Every {@{$[a,b+1/n)$}@} is in {@{$\mathcal B(\mathbb R)$}@}, so {@{$[a,b]=\bigcap_{n\ge1}[a,b+1/n)$}@} is {@{Borel by countable intersection}@}; similarly, {@{$(-\infty,a]=\bigcup_{n\ge1}(a-n,a]$}@} is {@{Borel by countable union}@}.
> - (c) Let {@{$\mathcal B_2=\sigma(\{(-\infty,b]:b\in\mathbb R\})$}@}. Then {@{$(-\infty,b)=\bigcap_{n\ge1}(-\infty,b+1/n]$}@} belongs to {@{$\mathcal B_2$}@}, and therefore each half-open interval satisfies {@{$[a,b)=(-\infty,b)\setminus(-\infty,a)$}@}; so {@{$\mathcal B(\mathbb R)\subseteq\mathcal B_2$}@}. Also each half-line satisfies {@{$(-\infty,b]=(b,\infty)^c$}@}, so {@{the sigma-algebra generated by open sets contains $\mathcal B_2$}@}. Conversely, {@{every open set is a countable union of disjoint open intervals}@}, and each interval {@{$(a,b)$}@} can be written as {@{$\bigcup_{n\ge1}[a+1/n,b)$}@}; thus every open set is Borel. Therefore all three generating families yield the same {@{Borel sigma-algebra}@}.

> Calculations with continuous distributions.
>
> - (a) If a lightbulb lifetime is exponentially distributed with parameter $\lambda>0$, compute the probability that it lasts between $2$ and $3$ years, and find the half-life.
> - (b) Show that the exponential distribution is memoryless.
> - (c) For the continuous Laplace-$2$ distribution with density $f(x)=c_\lambda e^{-|x|/\lambda}$, determine $c_\lambda$, the CDF, and $P([-1,1])$.
>
> Solution:
>
> - (a) For the exponential law with density {@{$\lambda e^{-\lambda x}\mathbf1_{(0,\infty)}(x)$}@}, {@{$P([2,3])=\int_2^3\lambda e^{-\lambda x}\,dx=e^{-2\lambda}-e^{-3\lambda}$}@}; when {@{$\lambda=1$}@}, this becomes {@{$e^{-2}(1-e^{-1})\approx0.0855$}@}. The half-life {@{$t_{1/2}$}@} solves {@{$P(X\ge t_{1/2})=e^{-\lambda t_{1/2}}=1/2$}@}, so {@{$t_{1/2}=\dfrac{\ln 2}{\lambda}$}@} by {@{solving the survival equation}@}.
> - (b) Because {@{$(s+t,\infty)\subseteq(s,\infty)$}@}, conditional probability gives {@{$P(X>s+t\mid X>s)=\dfrac{e^{-\lambda(s+t)}}{e^{-\lambda s}}=e^{-\lambda t}=P(X>t)$}@}; this is the {@{memoryless property}@}.
> - (c) Normalization yields {@{$1=2c_\lambda\int_0^{\infty}e^{-x/\lambda}\,dx=2c_\lambda\lambda$}@}, so {@{$c_\lambda=1/(2\lambda)$}@}. The CDF is {@{piecewise because the density is symmetric about $0$ but changes formula at the origin}@}: {@{$F(x)=\frac12e^{x/\lambda}$ for $x\le0$ and $F(x)=1-\frac12e^{-x/\lambda}$ for $x>0$}@}. Hence {@{$P([-1,1])=F(1)-F(-1)=1-e^{-1/\lambda}$}@}.

> On probability density functions.
>
> - (a) For which $k\in\mathbb R$, if any, is $f(x)=x^{-k}\mathbf1_{[1,\infty)}(x)$ a probability density function?
> - (b) Give a density $f$ such that $c\sqrt f$ cannot be a density for any $c>0$.
> - (c) For $f(x)=2|x|(1-x^2)\mathbf1_{\{|x|\le1\}}$, find the cumulative distribution function and compute $P(( -\infty,-1/2))$ and $P([-1/2,1/2])$.
>
> Solution:
>
> - (a) Integrability requires {@{$k>1$}@}, and then {@{$\int_1^{\infty}x^{-k}\,dx=1/(k-1)$}@}; making this equal to {@{$1$}@} forces the unique choice {@{$k=2$}@}. Thus {@{the only exponent producing a PDF is $k=2$}@}.
> - (b) Taking exactly that density {@{$f(x)=x^{-2}\mathbf1_{[1,\infty)}(x)$}@} works, because {@{$\sqrt f(x)=x^{-1}\mathbf1_{[1,\infty)}(x)$}@} is {@{not integrable on $[1,\infty)$}@}, so {@{no positive constant can normalize $c\sqrt f$ into a density}@}.
> - (c) The support is {@{$[-1,1]$}@}; integrating separately on the negative and positive halves gives the CDF {@{$F(x)=0$ for $x<-1$, $F(x)=\dfrac{(1-x^2)^2}{2}$ for $-1\le x<0$, $F(x)=1-\dfrac{(1-x^2)^2}{2}$ for $0\le x<1$, and $F(x)=1$ for $x\ge1$}@}. Since the law has a density, {@{single points have probability $0$}@}, so {@{$P(( -\infty,-1/2))=F(-1/2)=\dfrac{9}{32}$}@} and {@{$P([-1/2,1/2])=F(1/2)-F(-1/2)=\dfrac{7}{16}$}@}.
