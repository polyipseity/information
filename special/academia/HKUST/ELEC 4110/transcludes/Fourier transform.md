# Fourier transform

## definition

The Fourier transform of {@{a complex-valued function $f(x)$ on the real line}@}, is {@{the complex valued function ${\widehat {f} }(\xi )$}@}, defined by the integral<sup>[\[1\]](#^ref-1)</sup> <!--SR:!2026-02-14,69,339!2026-02-19,74,339-->

> {@{Fourier transform}@}
>
> {@{$${\widehat {f} }(\xi )=\int _{-\infty }^{\infty }f(x)\ e^{-i2\pi \xi x}\,dx,\quad \forall \xi \in \mathbb {R} \,. \qquad \tag{Eq.1}$$}@} <!--SR:!2026-02-23,77,339!2026-02-24,78,339-->

In this case {@{$f(x)$}@} is {@{\(Lebesgue\) integrable over the whole real line}@}, i.e., the above integral {@{converges to a continuous function ${\widehat {f} }(\xi )$ at all $\xi$}@} \(decaying to {@{zero as $\xi \to \infty$}@}\). <!--SR:!2026-03-06,87,358!2026-02-28,81,339!2026-02-28,82,358!2026-03-04,85,361-->

However, the Fourier transform can also be defined for {@{\(generalized\) functions}@} for which {@{the [Lebesgue integral](../../../../../general/eng/Lebesgue%20integral.md) __[Eq.1](#math%20Eq.1)__ does not make sense}@}.<sup>[\[2\]](#^ref-2)</sup> Interpreting {@{the integral [suitably](../../../../../general/eng/weak%20topology.md)}@} \(e.g. as an {@{[improper integral](../../../../../general/eng/improper%20integral.md) for [locally integrable](../../../../../general/eng/locally%20integrable.md) functions}@}\) extends {@{the Fourier transform to functions}@} that are {@{not necessarily integrable over the whole real line}@}. More generally, the Fourier transform also applies to {@{[generalized functions](../../../../../general/eng/generalized%20function.md)}@} like {@{the [Dirac delta](../../../../../general/eng/Dirac%20delta.md) \(and all other [tempered distributions](../../../../../general/eng/tempered%20distributions.md#tempered%20distributions)\)}@}, in which case it is defined by {@{duality rather than an integral}@}.<sup>[\[3\]](#^ref-3)</sup> <!--SR:!2026-02-23,77,339!2026-02-22,76,339!2026-03-06,86,361!2026-02-08,64,319!2026-03-07,87,361!2026-02-19,74,339!2026-02-24,78,339!2026-03-12,92,361!2026-03-09,89,361-->

First introduced in {@{[Fourier's](../../../../../general/eng/Joseph%20Fourier.md) _Analytical Theory of Heat_.}@},<sup>[\[4\]](#^ref-4)</sup><sup>[\[5\]](#^ref-5)</sup><sup>[\[6\]](#^ref-6)</sup><sup>[\[7\]](#^ref-7)</sup> {@{the corresponding inversion formula for "[sufficiently nice](../../../../../general/eng/Fourier%20inversion%20theorem.md#conditions%20on%20the%20function)" functions}@} is given by {@{the [Fourier inversion theorem](../../../../../general/eng/Fourier%20inversion%20theorem.md)}@}, i.e., <!--SR:!2026-02-17,72,339!2026-02-05,61,319!2026-02-24,78,339-->

> {@{Inverse transform}@}
>
> {@{$$f(x)=\int _{-\infty }^{\infty }{\widehat {f} }(\xi )\ e^{i2\pi \xi x}\,d\xi ,\quad \forall \ x\in \mathbb {R} \,. \qquad \tag{Eq.2}$$}@} <!--SR:!2026-02-28,81,339!2026-03-11,91,361-->

{@{The functions $f$ and ${\widehat {f} }$}@} are referred to as {@{a __Fourier transform pair__}@}.<sup>[\[8\]](#^ref-8)</sup>  {@{A common notation for designating transform pairs}@} is<!-- markdown separator -->__:__<sup>[\[9\]](#^ref-9)</sup> {@{$$f(x)\ {\stackrel {\mathcal {F} }{\longleftrightarrow } }\ {\widehat {f} }(\xi ).$$}@} For example, {@{the Fourier transform of the delta function}@} is {@{the constant function $1$}@}: {@{$$\delta (x)\ {\stackrel {\mathcal {F} }{\longleftrightarrow } }\ 1.$$}@} <!--SR:!2026-02-16,71,334!2026-02-06,62,319!2026-03-06,87,358!2026-02-24,78,339!2026-02-16,71,339!2026-02-26,80,358!2026-03-07,87,361-->

### angular frequency \(_ω_\)

When {@{the independent variable \($x$\)}@} represents {@{_time_ \(often denoted by $t$\)}@}, {@{the transform variable \($\xi$\)}@} represents {@{[frequency](../../../../../general/eng/frequency.md) \(often denoted by $f$\)}@}. For example, if time has {@{the unit [second](../../../../../general/eng/second.md)}@}, then frequency has {@{the unit [hertz](../../../../../general/eng/hertz.md)}@}. {@{The transform variable}@} can also be written in terms of {@{[angular frequency](../../../../../general/eng/angular%20frequency.md), $\omega =2\pi \xi$}@}, with {@{the unit [radian](../../../../../general/eng/radian.md) per second}@}. <!--SR:!2026-03-08,88,358!2026-02-21,75,339!2026-03-05,86,361!2026-03-10,90,361!2026-02-08,64,334!2026-02-23,77,339!2026-02-14,69,339!2026-02-17,72,339!2026-03-07,87,361-->

{@{The substitution $\xi ={\tfrac {\omega }{2\pi } }$}@} into __[Eq.1](#math%20Eq.1)__ produces {@{this convention}@}, where function ${\widehat {f} }$ is {@{relabeled ${\widehat {f} }_{1}:$}@} {@{$${\begin{aligned}{\widehat {f} }_{3}(\omega )&\triangleq \int _{-\infty }^{\infty }f(x)\cdot e^{-i\omega x}\,dx={\widehat {f} }_{1}\left({\tfrac {\omega }{2\pi } }\right),\\f(x)&={\frac {1}{2\pi } }\int _{-\infty }^{\infty }{\widehat {f} }_{3}(\omega )\cdot e^{i\omega x}\,d\omega .\end{aligned} }$$}@} Unlike {@{the __[Eq.1](#math%20Eq.1)__ definition}@}, the Fourier transform is {@{no longer a [unitary transformation](../../../../../general/eng/unitary%20transformation.md)}@}, and there is {@{less symmetry between the formulas for the transform and its inverse}@}. {@{Those properties}@} are restored by {@{splitting the $2\pi$ factor evenly between the transform and its inverse}@}, which leads to {@{another convention}@}: {@{$${\begin{aligned}{\widehat {f} }_{2}(\omega )&\triangleq {\frac {1}{\sqrt {2\pi } } }\int _{-\infty }^{\infty }f(x)\cdot e^{-i\omega x}\,dx={\frac {1}{\sqrt {2\pi } } }\ \ {\widehat {f} }_{1}\left({\tfrac {\omega }{2\pi } }\right),\\f(x)&={\frac {1}{\sqrt {2\pi } } }\int _{-\infty }^{\infty }{\widehat {f} }_{2}(\omega )\cdot e^{i\omega x}\,d\omega .\end{aligned} }$$}@} {@{Variations of all three conventions}@} can be created by {@{conjugating the complex-exponential [kernel](../../../../../general/eng/integral%20kernel.md)}@} of {@{both the forward and the reverse transform}@}. {@{The signs}@} must be {@{opposites}@}. <!--SR:!2026-02-23,77,339!2026-03-09,89,361!2026-02-22,76,339!2026-02-15,70,339!2026-02-27,81,361!2026-02-19,74,339!2026-02-14,69,339!2026-03-08,88,361!2026-03-11,91,358!2026-03-03,84,361!2026-03-05,86,361!2026-03-08,88,358!2026-02-28,82,361!2026-02-15,70,339!2026-02-17,72,339!2026-03-03,84,358-->

### Lebesgue integrable functions

- See also: ::@:: [Lp space § Lp spaces and Lebesgue integrals](../../../../../general/eng/Lp%20space.md#Lp%20spaces%20and%20Lebesgue%20integrals) <!--SR:!2026-02-17,72,339!2026-02-20,74,339-->

{@{A [measurable function](../../../../../general/eng/measurable%20function.md) $f:\mathbb {R} \to \mathbb {C}$}@} is {@{called \(Lebesgue\) integrable}@} if {@{the [Lebesgue integral](../../../../../general/eng/Lebesgue%20integral.md) of its absolute value is finite}@}: {@{$$\|f\|_{1}=\int _{\mathbb {R} }|f(x)|\,dx<\infty .$$}@} If {@{$f$ is Lebesgue integrable}@} then {@{the Fourier transform, given by __[Eq.1](#math%20Eq.1)__}@}, is {@{well-defined for all $\xi \in \mathbb {R}$}@}.<sup>[\[10\]](#^ref-10)</sup> Furthermore, {@{${\widehat {f} }\in L^{\infty }\cap C_{0}(\mathbb {R} )$ is bounded}@}, {@{[uniformly continuous](../../../../../general/eng/uniformly%20continuous.md)}@} and {@{\(by the [Riemann–Lebesgue lemma](../../../../../general/eng/Riemann–Lebesgue%20lemma.md)\) [vanishing at infinity](../../../../../general/eng/vanish%20at%20infinity.md)}@}. Here {@{$C_{0}(\mathbb {R} )$}@} denotes {@{the space of continuous functions on $\mathbb {R}$}@} that {@{approach 0 as x approaches positive or negative infinity}@}. <!--SR:!2026-03-07,87,358!2026-02-15,70,339!2026-02-23,77,339!2026-02-08,65,341!2026-02-19,74,339!2026-03-08,88,361!2026-03-05,86,361!2026-02-10,63,341!2026-03-04,85,361!2026-02-27,80,339!2026-03-06,87,361!2026-03-01,82,339!2026-03-09,89,361-->

{@{The space $L^{1}(\mathbb {R} )$}@} is {@{the space of measurable functions}@} for which {@{the norm $\|f\|_{1}$ is finite}@}, modulo {@{the [equivalence relation](../../../../../general/eng/equivalence%20class.md) of equality [almost everywhere](../../../../../general/eng/almost%20everywhere.md)}@}. {@{The Fourier transform on $L^{1}(\mathbb {R} )$}@} is {@{[one-to-one](../../../../../general/eng/bijection,%20injection%20and%20surjection.md)}@}. However, there is {@{no easy characterization of the image}@}, and thus {@{no easy characterization of the inverse transform}@}. In particular, {@{__[Eq.2](#math%20Eq.2)__ is no longer valid}@}, as it was stated {@{only under the hypothesis that $f(x)$ was "sufficiently nice"}@} \(e.g., $f(x)$ {@{[decays with all derivatives](../../../../../general/eng/Schwartz%20function.md)}@}\). <!--SR:!2026-02-28,82,358!2026-03-11,91,361!2026-02-26,79,339!2026-03-05,86,361!2026-03-06,87,361!2026-03-10,90,361!2026-03-12,92,361!2026-02-25,78,339!2026-02-22,77,356!2026-02-16,71,339!2026-03-01,82,339-->

While {@{__[Eq.1](#math%20Eq.1)__ defines the Fourier transform}@} for {@{\(complex-valued\) functions in $L^{1}(\mathbb {R} )$}@}, it is {@{not well-defined for other integrability classes}@}, most importantly {@{the space of [square-integrable functions](../../../../../general/eng/square-integrable%20function.md) $L^{2}(\mathbb {R} )$}@}. For example, {@{the function $f(x)=(1+x^{2})^{-1/2}$}@} is {@{in $L^{2}$ but not $L^{1}$}@} and therefore {@{the Lebesgue integral __[Eq.1](#math%20Eq.1)__ does not exist}@}. However, the Fourier transform on {@{the dense subspace $L^{1}\cap L^{2}(\mathbb {R} )\subset L^{2}(\mathbb {R} )$}@} admits {@{a unique continuous extension}@} to {@{a [unitary operator](../../../../../general/eng/unitary%20operator.md) on $L^{2}(\mathbb {R} )$}@}. {@{This extension is important}@} in part because, unlike {@{the case of $L^{1}$}@}, the Fourier transform is {@{an [automorphism](../../../../../general/eng/automorphism.md) of the space $L^{2}(\mathbb {R} )$}@}. <!--SR:!2026-10-19,258,334!2026-02-27,81,361!2026-02-19,74,339!2026-02-24,78,339!2026-07-18,165,319!2026-03-05,86,361!2026-02-22,76,339!2026-02-23,77,339!2026-03-01,83,361!2026-03-06,87,358!2026-03-09,89,361!2026-02-23,77,339!2026-03-03,84,361-->

In such cases, the Fourier transform can be {@{obtained explicitly}@} by {@{[regularizing](../../../../../general/eng/regularization%20(mathematics).md) the integral}@}, and then {@{passing to a limit}@}. In practice, {@{the integral}@} is often {@{regarded as an [improper integral](../../../../../general/eng/improper%20integral.md) instead of a proper Lebesgue integral}@}, but sometimes {@{for convergence}@} one needs to {@{use [weak limit](../../../../../general/eng/weak%20limit.md) or [principal value](../../../../../general/eng/Cauchy%20principal%20value.md)}@} instead of {@{the \(pointwise\) limits implicit in an improper integral}@}. {@{[Titchmarsh \(1986\)](#CITEREFTitchmarsh1986) and [Dym & McKean \(1985\)](#CITEREFDymMcKean1985)}@} each gives {@{three rigorous ways of extending the Fourier transform}@} to {@{square integrable functions using this procedure}@}. {@{A general principle}@} in {@{working with the $L^{2}$ Fourier transform}@} is that {@{Gaussians are dense in $L^{1}\cap L^{2}$}@}, and {@{the various features of the Fourier transform}@}, such as {@{its unitarity}@}, are {@{easily inferred for Gaussians}@}. {@{Many of the properties of the Fourier transform}@} can then be proven from {@{two facts about Gaussians}@}:<sup>[\[11\]](#^ref-11)</sup> \(annotation: 2 items: {@{own Fourier transform, Gaussian integral}@}\) <!--SR:!2026-03-11,91,361!2026-03-12,92,361!2026-03-01,82,339!2026-02-22,76,339!2026-02-27,81,358!2026-03-12,92,361!2026-02-14,69,339!2026-02-19,74,339!2026-03-04,85,356!2026-03-10,90,361!2026-03-12,92,361!2026-03-04,85,361!2026-02-19,74,339!2026-03-11,91,361!2026-02-23,77,339!2026-02-23,77,339!2026-02-14,69,339!2026-03-08,88,358!2026-02-14,69,339!2026-03-07,87,361-->

- that $e^{-\pi x^{2} }$ ::@:: is its own Fourier transform; and <!--SR:!2026-02-06,62,319!2026-02-27,80,339-->
- that the [Gaussian integral](../../../../../general/eng/Gaussian%20integral.md) ::@:: $\int _{-\infty }^{\infty }e^{-\pi x^{2} }\,dx=1$. <!--SR:!2026-02-16,71,339!2026-03-12,92,361-->

{@{A feature of the $L^{1}$ Fourier transform}@} is that it is {@{a homomorphism of Banach algebras}@} from {@{$L^{1}$ equipped with the convolution operation}@} to {@{the Banach algebra of continuous functions}@} under {@{the $L^{\infty }$ \(supremum\) norm}@}. {@{The conventions}@} chosen in this article are those of {@{[harmonic analysis](../../../../../general/eng/harmonic%20analysis.md)}@}, and are characterized as {@{the unique conventions}@} such that {@{the Fourier transform}@} is both {@{[unitary](../../../../../general/eng/unitary%20operator.md) on _L_<sup>2</sup> and an algebra homomorphism from _L_<sup>1</sup> to _L_<sup>∞</sup>}@}, without {@{renormalizing the Lebesgue measure}@}.<sup>[\[12\]](#^ref-12)</sup> <!--SR:!2026-03-06,86,358!2026-02-24,78,339!2026-02-16,71,339!2026-03-05,86,361!2026-03-07,87,361!2026-03-07,87,361!2026-03-06,86,358!2026-02-23,77,339!2026-03-05,86,361!2026-02-22,77,356!2026-02-27,80,339-->

## properties

Let {@{$f(x)$ and $h(x)$}@} represent {@{_integrable functions_ [Lebesgue-measurable](../../../../../general/eng/Lebesgue-measurable.md) on the real line}@} satisfying: {@{$$\int _{-\infty }^{\infty }|f(x)|\,dx<\infty .$$ \(annotation: Also includes $h(x)$.\)}@} We denote {@{the Fourier transforms of these functions}@} as {@{${\widehat {f} }(\xi )$ and ${\widehat {h} }(\xi )$ respectively}@}. <!--SR:!2026-02-16,71,339!2026-03-12,92,361!2026-03-08,88,361!2026-02-12,68,334!2026-03-06,87,358-->

### basic properties

The Fourier transform has {@{the following basic properties}@}:<sup>[\[15\]](#^ref-15)</sup> <!--SR:!2026-02-17,72,339-->

#### linearity

\(annotation: linearity\) ::@:: $$a\ f(x)+b\ h(x)\ \ {\stackrel {\mathcal {F} }{\Longleftrightarrow } }\ \ a\ {\widehat {f} }(\xi )+b\ {\widehat {h} }(\xi );\quad \ a,b\in \mathbb {C}$$ <!--SR:!2026-02-22,76,339!2026-02-05,61,330-->

#### time shifting

\(annotation: time shifting\) ::@:: $$f(x-x_{0})\ \ {\stackrel {\mathcal {F} }{\Longleftrightarrow } }\ \ e^{-i2\pi x_{0}\xi }\ {\widehat {f} }(\xi );\quad \ x_{0}\in \mathbb {R}$$ <!--SR:!2026-03-09,89,361!2026-08-12,190,338-->

#### frequency shifting

\(annotation: frequency shifting\) ::@:: $$e^{i2\pi \xi _{0}x}f(x)\ \ {\stackrel {\mathcal {F} }{\Longleftrightarrow } }\ \ {\widehat {f} }(\xi -\xi _{0});\quad \ \xi _{0}\in \mathbb {R}$$ <!--SR:!2026-02-15,70,339!2026-02-26,80,358-->

#### time scaling

\(annotation:  time scaling\) ::@:: $$f(ax)\ \ {\stackrel {\mathcal {F} }{\Longleftrightarrow } }\ \ {\frac {1}{|a|} }{\widehat {f} }\left({\frac {\xi }{a} }\right);\quad \ a\neq 0$$ <!--SR:!2026-02-25,78,339!2026-02-20,74,339-->

{@{The case $a=-1$}@} leads to {@{the _time-reversal property_}@}: {@{$$f(-x)\ \ {\stackrel {\mathcal {F} }{\Longleftrightarrow } }\ \ {\widehat {f} }(-\xi )$$}@} <!--SR:!2026-02-15,70,339!2026-03-06,87,358!2026-02-27,80,339-->

### convolution theorem

### differentiation

Suppose {@{_f_\(_x_\)}@} is {@{differentiable [almost everywhere](../../../../../general/eng/almost%20everywhere.md)}@}, and both {@{_f_ and its derivative _f′_ are integrable}@} \(in {@{$L^{1}(\mathbb {R} )$}@}\). ### converting Fourier transform pairs between conventions

\(annotation: applies to this section\) Using {@{the steps below}@}, one can {@{convert Fourier transform pairs between different conventions}@}: \(annotation: 3 ordered items: {@{recall and apply relations → handle special expressions → simplify}@}\) <!--SR:!2026-03-12,92,361!2026-03-07,87,358!2026-05-15,124,318-->

1. recall and apply relations ::@:: Recall the two forward and backward relations between the source and target convention. For example, to convert in between unitary convention using ordinary frequency and non-unitary convention using angular frequency, recall: $$\begin{aligned} \widehat f_1(\xi) & = \widehat f_3(2\pi \xi) \\ \widehat f_3(\omega) & = \widehat f_1\left(\frac \omega {2\pi} \right) \,. \end{aligned}$$ <p> Label all Fourier transforms with the original convention to avoid confusion, e.g. replacing $\widehat f$ with $\widehat f_1$. Then apply the relations until all Fourier transforms are converted, which is necessary because a Fourier transform of a function may be written in terms of Fourier transforms of simpler functions. <!--SR:!2026-06-24,141,407!2026-06-24,141,407-->
2. handle special expressions ::@:: Apart from Fourier transforms, some special functions also transform with additional multiplying prefactors and/or additional factors in the argument. Notable examples: <p> - Dirac delta scaling: $\delta^{(n)} \left(\frac {\omega} {2\pi} \right) = \lvert 2\pi \rvert^{n + 1} \delta^{(n)}(\omega) = (2\pi)^{n + 1} \delta^{(n)}(\omega)$ and $\delta^{(n)}(2 \pi \xi) = \frac {\delta^{(n)}(\xi)} {\lvert 2\pi \rvert^{n + 1} } = \frac {\delta^{(n)}(\xi)} {(2\pi)^{n + 1} }$ <br/> - multiplication as convolution: $f(x)g(x)\, {\stackrel {\mathcal {F} }{\Longleftrightarrow } } \left({\widehat {f} }*{\widehat {g} }\right)(\xi )\, \quad ; \quad {\frac {1}{\sqrt {2\pi } } }\left({\widehat {f} }*{\widehat {g} }\right)(\omega )\, \quad ; \quad {\frac {1}{2\pi } }\left({\widehat {f} }*{\widehat {g} }\right)(\omega )\,$ <!--SR:!2026-06-23,140,407!2026-05-18,104,387-->
3. simplify ::@:: Simplify the results. Notable examples: <p> - differentiation: $\frac {\mathrm d} {\mathrm d(ax)} = \frac {\mathrm d} {a \, \mathrm dx} = \frac 1 a \frac {\mathrm d} {\mathrm dx}$ <br/> - even function: $f(-x) = f(x)$, e.g. $\cos$, $\operatorname{rect}$, $\operatorname{sinc}$, $\operatorname{tri}$, etc. <!--SR:!2026-06-23,140,407!2026-06-16,136,407-->

### deriving the dual of Fourier transform pairs

\(annotation: applies to this section\) Using {@{the steps below}@}, one can {@{derive the dual of Fourier transform pairs}@}: \(annotation: 3 ordered items: {@{recall and apply relations → replace Fourier transforms → simplify}@}\) <!--SR:!2026-03-12,92,361!2026-03-01,82,339!2026-09-30,242,339-->

1. recall and apply relations ::@:: Recall: $${\widehat {f} }_{n}(x)\, {\stackrel {\mathcal {F} }{\Longleftrightarrow } } {\widehat {f} }_{1}(x)\ {\stackrel { {\mathcal {F} }_{1} }{\longleftrightarrow } }\ f(-\xi )\, \quad ; \quad {\widehat {f} }_{2}(x)\ {\stackrel { {\mathcal {F} }_{2} }{\longleftrightarrow } }\ f(-\omega )\, \quad ; \quad {\widehat {f} }_{3}(x)\ {\stackrel { {\mathcal {F} }_{3} }{\longleftrightarrow } }\ 2\pi f(-\omega )\,.$$ Apply the above relation, which should _automatically_ replace the time or frequency variables. <p> Note when negating the argument $-x$, $-\xi$, or $-\omega$, this includes $x$, $\xi$, or $\omega$ respectively in $\mathrm dx$, $\mathrm d\xi$, or $\mathrm d\omega$. Due to this, you should use Leibniz's notation $\frac {\mathrm d} {\mathrm dx}$ for differentiation. <!--SR:!2026-06-18,138,407!2026-06-24,141,407-->
2. replace Fourier transforms ::@:: Currently, the left hand side may (or may not) contain the Fourier transform of the function while the right hand side may (or may not) contain the original function. If they do not appear, which happens if the original function and its Fourier transform are already in their _expanded_ form, then no further action is needed. If they appear, we want to rewrite these expressions. <p> To do so, recall the relation. Then, replace every Fourier transform in the _exact_ form of the LHS of the relation with $f(g(x))$, where $g(x)$ is the expression after _pattern matching_ on the input argument $x$; and replace every original function in the _exact_ form of the RHS of the relation with $\widehat f(g(\xi))$, where $g(\xi)$ is the expression after _pattern matching_ on the input argument $\xi$. <p> For example, the non-unitary transform in angular frequency has the relation: $$\widehat f_3(x) \longleftrightarrow 2\pi f(-\xi) \,.$$ For every Fourier transform on the left hand side, _pattern match_ with the _exact_ expression $\widehat f_3(x)$. This additionally yields a matched expression for $x$, which we then denote as $g(x)$. Then replace the entire matched expression with $f(g(x))$. For every original function on the right had side, _pattern match_ with the _exact_ expression $2\pi f(-\xi)$. This additionally yields a matched expression for $\xi$, which we then denote as $g(\xi)$. Then replace the entire matched expression with $\widehat f_3(g(\xi))$. <!--SR:!2026-06-24,141,407!2026-06-23,140,407-->
3. simplify ::@:: Simplify the results. Optionally, move constant factors around by linearity. Notable examples: <p> - convolution, time reversal: $(f(-u) * g(-u))(x) = (f(u) * g(u))(-x)$; the notation used here should be interpreted carefully... <br/> - differentiation: $\frac {\mathrm d} {\mathrm d(ax)} = \frac {\mathrm d} {a \, \mathrm dx} = \frac 1 a \frac {\mathrm d} {\mathrm dx}$ <br/> - even function: $f(-x) = f(x)$, e.g. $\cos$, $\operatorname{rect}$, $\operatorname{sinc}$, $\operatorname{tri}$, etc. <!--SR:!2026-06-24,141,407!2026-06-15,135,407-->

## notes

1. {@{Sentence structure}@} is often {@{sufficient to distinguish the intended meaning}@}: for example, {@{"Apply the Fourier transform to \[an input\]"}@} refers to {@{the operation}@}, whereas {@{"The Fourier transform of \[an input\]"}@} refers to {@{its output}@}. <a id="^ref-1"></a>^ref-1 <!--SR:!2026-03-08,88,361!2026-02-28,81,339!2026-02-19,74,339!2026-02-16,71,339!2026-02-18,73,339!2026-03-03,84,361-->

2. Depending on {@{the application}@} {@{a [Lebesgue integral](../../../../../general/eng/Lebesgue%20integral.md), [distributional](../../../../../general/eng/distribution%20(mathematics).md), or other approach}@} may be {@{most appropriate}@}. <a id="^ref-2"></a>^ref-2 <!--SR:!2026-03-04,85,361!2026-02-19,73,319!2026-02-24,78,339-->

3. [Vretblad \(2000\)](#CITEREFVretblad2000) provides {@{solid justification for these formal procedures}@} without going too deeply into {@{[functional analysis](../../../../../general/eng/functional%20analysis.md) or the [theory of distributions](../../../../../general/eng/distribution%20(mathematics).md)}@}. <a id="^ref-3"></a>^ref-3 <!--SR:!2026-02-18,73,339!2026-02-24,78,339-->

4. In {@{[relativistic quantum mechanics](../../../../../general/eng/relativistic%20quantum%20mechanics.md)}@} one encounters {@{vector-valued Fourier transforms}@} of {@{multi-component wave functions}@}. In {@{[quantum field theory](../../../../../general/eng/quantum%20field%20theory.md),}@} {@{operator-valued Fourier transforms}@} of {@{operator-valued functions of spacetime}@} are {@{in frequent use}@}, see for example [Greiner & Reinhardt \(1996\)](#CITEREFGreinerReinhardt1996). <a id="^ref-4"></a>^ref-4 <!--SR:!2026-03-01,82,339!2026-02-11,67,330!2026-02-16,71,339!2026-02-16,71,339!2026-02-17,72,339!2026-03-12,92,361!2026-02-22,76,339-->

5. {@{A possible source of confusion}@} is {@{the [frequency-shifting property](#frequency%20shifting)}@}; i.e. {@{the transform of function $f(x)e^{-i2\pi \xi _{0}x}$}@} is {@{${\widehat {f} }(\xi +\xi _{0})$}@}.  The value of {@{this function at  $\xi =0$}@}  is {@{${\widehat {f} }(\xi _{0})$}@}, meaning that {@{a frequency $\xi _{0}$}@} has been {@{shifted to zero}@} \(also see {@{[Negative frequency](../../../../../general/eng/negative%20frequency.md#simplifying%20the%20Fourier%20transform)}@}\). <a id="^ref-5"></a>^ref-5 <!--SR:!2026-03-08,88,361!2026-03-07,87,361!2026-02-24,78,339!2026-02-24,78,339!2026-03-04,85,361!2026-03-01,83,361!2026-03-06,86,361!2026-02-18,73,339!2026-02-25,79,358-->

6. {@{The operator $U\left({\frac {1}{2\pi } }{\frac {d}{dx} }\right)$}@} is defined by {@{replacing $x$ by ${\frac {1}{2\pi } }{\frac {d}{dx} }$}@} in {@{the [Taylor expansion](../../../../../general/eng/Taylor%20series.md) of $U(x)$}@}. <a id="^ref-6"></a>^ref-6
7. Up to {@{an imaginary constant factor}@} whose {@{magnitude}@} depends on {@{what Fourier transform convention is used}@}. <a id="^ref-7"></a>^ref-7 <!--SR:!2026-02-26,79,339!2026-03-10,90,361!2026-03-07,87,358!2026-03-10,90,358!2026-02-23,77,339!2026-03-06,87,361-->

8. The direct command {@{`fourier transform of cos(6*pi*t) exp(−pi*t^2)`}@} would also {@{work for Wolfram Alpha}@}, although {@{the options for the convention}@} \(see {@{[Fourier transform § Other conventions](#other%20conventions)}@}\) must be {@{changed away from the default option}@}, which is actually equivalent to {@{`integrate cos(6*pi*t) exp(−pi*t^2) exp(i*omega*t) /sqrt(2*pi) from -inf to inf`}@}. <a id="^ref-8"></a>^ref-8 <!--SR:!2026-03-05,86,358!2026-02-17,72,339!2026-03-05,86,361!2026-03-09,89,361!2026-02-23,77,339!2026-03-10,90,358-->
